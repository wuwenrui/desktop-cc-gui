# 破产业务小助手

## 技能定位
全局技能，已掌握破产法核心知识体系。用户只需提供当次个性化任务资料，即可进入具体事务工作区进行个性化处理。原始学习资料可随时删除，知识已固化在本技能中。

## 一、法律基础体系

### 1.1 核心法律
- **《中华人民共和国企业破产法》**（2006年8月27日通过，2007年6月1日施行）
- 三大程序：破产清算、和解、重整（程序间可依法转换）
- 管理人制度：由律师事务所、会计师事务所、破产清算事务所等中介机构担任
- 破产费用和共益债务（第41-43条）
- 法律责任（第125-131条）

### 1.2 管辖规则
- 地域管辖：债务人住所地法院（主要办事机构所在地；无办事机构的以注册地为准）
- 级别管辖：基层法院管辖县级/区级工商登记企业；中院管辖地市级以上工商登记企业、纳入国家计划调整案件、上市公司破产重整案件、执转破案件
- 受理后专属管辖：有关债务人的民事诉讼只能向受理破产申请的人民法院提起（第21条）
- 关联企业实质合并：由核心控制企业住所地法院管辖；核心不明确则由主要财产所在地法院管辖
- 执转破管辖：以中院管辖为原则，基层法院为例外

### 1.3 申请与受理
- 申请主体：债务人、债权人、清算义务人（解散清算转破产）
- 债权人申请条件：只需证明不能清偿到期债务，不需证明资不抵债
- 债务人申请条件：需证明资不抵债，但不需要专项审计，提供资产负债比即可
- 撤回申请：仅可在受理前撤回（第9条）
- 债务人对申请有异议的，应在收到通知后7日内提出；法院应在异议期满后10日内裁定是否受理

### 1.4 管理人
- 管理人由人民法院指定（第22条）
- 职责（第25条）：接管债务人的财产印章账簿文书、调查财产状况、决定内部管理事务、管理和处分财产、代表债务人参加诉讼仲裁等
- 管理人应当勤勉尽责，忠实执行职务（第27条）
- 管理人无正当理由不得辞去职务（第29条）
- 管理人定位：司法备位、市场主位、政府配位

## 二、债权审核与异议制度

### 2.1 债权申报
- 人民法院受理破产申请后，债权人应在规定期限内申报债权（第44-56条）
- 未到期的债权，在破产申请受理时视为到期
- 附利息的债权自破产申请受理时起停止计息

### 2.2 债权审核流程
1. 债权人申报
2. 管理人调查
3. 管理人代表全体债权人认定性质与金额
4. 无争议→确认；有争议→破产延伸诉讼

### 2.3 债权清偿顺序（第113条）
| 顺序 | 内容 |
|------|------|
| 第一顺位 | 职工工资、医疗伤残补助抚恤费用、基本养老医疗保险费用、法定补偿金；董监高工资按职工平均工资计算 |
| 第二顺位 | 社会保险费用（除第一顺位外的）和税款 |
| 第三顺位 | 普通破产债权 |

### 2.4 债权清偿原则（《全国法院破产审判工作会议纪要》第28条）
- 人身损害赔偿优先于财产性债权
- 私法债权优先于公法债权
- 补偿性债权优先于惩罚性债权
- 人身损害赔偿参照职工债权顺序清偿（惩罚性赔偿除外）
- 剩余财产清偿顺序：民事惩罚性赔偿金 → 行政罚款 → 刑事罚金
- 关联企业不当关联形成的债权劣后于其他普通债权顺序清偿

### 2.5 个别清偿撤销权
- 第31条（受理前1年内）：无偿转让财产、明显不合理价格交易、对无担保债务提供担保、对未到期债务提前清偿、放弃债权
- 第32条（受理前6个月内）：债务人已具备破产原因仍对个别债权人清偿（使债务人财产受益的除外）
- **《破产法司法解释（二）》第15条**：经诉讼、仲裁、执行程序进行的个别清偿不可撤销
- 最高院（2016）最高法民申717号：银行在诉讼中主动扣划款项并获得生效判决确认的，属于"经过诉讼进行的个别清偿"，管理人不得撤销
- **例外**：债务人与债权人恶意串通损害其他债权人利益的

### 2.6 债权确认诉讼
- 对债权表记载的债权有异议的，可以向受理破产申请的人民法院提起诉讼（第58条）
- 异议人应在债权人会议核查结束后15日内提起
- 已经生效法律文书确定的债权，管理人应当予以确认。如认为有错误，应通过审判监督程序撤销

## 三、债务人财产制度

### 3.1 一般规定（第30条）
- 破产申请受理时属于债务人的全部财产
- 破产申请受理后至破产程序终结前债务人取得的财产

### 3.2 可撤销行为（第31、32条）
- 受理前1年内的5类可撤销行为（转让、交易、担保、提前清偿、放弃债权）
- 受理前6个月内的个别清偿

### 3.3 无效行为（第33条）
- 为逃避债务而隐匿、转移财产
- 虚构债务或承认不真实债务

### 3.4 出资追缴（第35条）
- 管理人应要求出资人缴纳所认缴的出资，不受出资期限限制
- 抽逃出资由管理人追回

### 3.5 追回权
- 董事、监事和高级管理人员利用职权获取的非正常收入和侵占的企业财产，管理人应当追回
- 因可撤销行为或无效行为取得的财产，管理人有权追回

## 四、职工债权认定

### 4.1 第一顺位清偿范围
- 职工工资
- 医疗、伤残补助、抚恤费用
- 应划入职工个人账户的基本养老保险、基本医疗保险费用
- 法律、行政法规规定的补偿金
- 董监高工资按职工平均工资计算

### 4.2 特殊规则
- 第132条：破产法公布之日前所欠职工债权在担保财产中优先于担保权人受偿
- 第三方垫付的职工债权，按垫付的职工债权性质清偿
- 欠薪保障基金垫付的，按第113条第1款第2项顺序清偿
- 住房公积金按职工工资性质清偿
- 破产程序应当依法保障企业职工合法权益（第6条），追究经营管理人员的法律责任

### 4.3 职工债权异议实务要点
- 审查关键：是否存在真实的劳动关系
- 公司实际控制人/管理者的职工债权需审慎认定
- 存在重大过错导致公司破产的管理者，其职工债权是否应予认定存在争议
- 应从劳动关系的实质要件判断，而非仅凭形式上的工资发放记录

## 五、破产重整

### 5.1 重整申请和期间
- 债务人、债权人、出资额占注册资本1/10以上的出资人均可申请重整
- 重整期间：自法院裁定重整之日起至重整程序终止
- 重整期间管理人或债务人负责管理财产和营业事务

### 5.2 重整计划的制定和批准
- 重整计划草案由管理人或债务人制定
- 分组表决：有担保债权组、职工债权组、税款债权组、普通债权组
- 法院强制批准应审慎适用，各反对组获得的清偿不低于清算利益

### 5.3 重整对象识别
- 需具有挽救价值和可能（《全国法院破产审判工作会议纪要》）
- 僵尸企业应清算出清，不宜重整

## 六、执行与破产衔接（执转破）

- 执行法院发现符合破产条件时，应及时询问当事人是否同意移送
- 移送后中止执行程序
- 破产受理后，执行法院应解除查封扣押冻结措施或移交处置权
- 破产受理法院可使用执行查控系统查控债务人财产
- 三个环节：决定程序 → 移送程序 → 审查处理程序

## 七、实务文书撰写规范

### 7.1 债权异议书
- 结构：标题→致管理人→引言（收到材料+异议对象）→分项异议（事实与理由）→结语→异议人签章→日期
- 语言：正式、严谨、法律文书风格
- 理由组织：法律依据 → 事实分析 → 推理 → 结论
- 每项异议独立成段，使用"一、二、三"编号

### 7.2 常见表述修订要点
- 法律文书中的"做"→"作"（作处理、作论述）
- 重复字修正（如"的的"→"的"）
- "与常规"→"与行业常规"
- 代词统一："该等"（法律文书规范用语）
- 逻辑一致性检查（如反对意见用了"应予认定"实为"不予认定"）

### 7.3 工程款优先权异议
- 工程造价审核争议：可申请重新审计（管理人委托第三方 vs 异议人另行委托）
- 存在关联关系的施工单位虚报工程量问题
- 合同偏差率异常（>15%需特别关注，>30%可能存在重大问题）
- 已取得生效法律文书的工程款债权需通过审判监督程序

## 八、工作流程

1. **接收任务** → 用户提供当次个性化任务资料路径
2. **学习阶段** → 读取所有提供的文件资料，理解案情
3. **知识激活** → 调取本技能知识库，与案情对照分析
4. **工作阶段** → 进入具体事务工作区，进行个性化处理
5. **副本原则** → 修改文稿时先建立副本，不动原稿
6. **输出成果** → 提交修订/润色/分析结果，说明修改要点

## 九、关联案件背景（天沃置业破产案）

- 债务人：淮南市天沃置业有限责任公司（开发"淮南市南山壹号"项目）
- 异议人：浙江诸暨康维投资合伙企业（有限合伙）
- 程序阶段：四债会（第四次债权人会议）
- 核心争议焦点：
  1. 新申报的农民工债权真实性
  2. 以天沃房产抵偿孔令全个人债务的性质认定
  3. 赵静（双重债权）和谢玉俊（逾期申报）的债权认定
  4. 工程款造价审核偏差问题（园林工程为例，偏差率>27%）
  5. 高巍和孔令全的职工债权认定
- 管理人认定标准不统一：相同付款条件下的不同债权利息认定不一致
- 关联人物：孔令全（实际控制人，涉及个人债务转嫁、以房抵债、关联交易）
