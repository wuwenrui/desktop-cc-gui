## 功能说明

本 Skill 用于处理**企业内部规章制度、管理办法、实施细则**等制度文件的审查法律事务，涵盖制度的合法性审查、合规风险分析、可执行性评估和修改建议。

## 〇、启动前置步骤——联网学习（每次启动必须执行）

**每次启动本 Skill 进行制度审查时，必须先完成以下联网学习步骤，不得跳过**：

### 0.1 识别制度涉及的领域
根据制度标题、内容涉及的管理事项，提取关键词（如：存货管理、行政管理、财务报销、车辆管理、考勤休假、印章管理、安全生产等）。

### 0.2 联网检索（至少3个方向）
根据制度领域，至少从以下方向中选择3个进行联网检索：
1. **法律法规动态**：检索该领域最新法规、修订动态（注意时效性）
2. **行业监管政策**：检索行业主管部门的最新规范性文件
3. **地方规定**：如企业位于特定省份，检索当地的地方性法规和国资监管要求
4. **上位制度依据**：检索是否有必须依据的上位法律、行政法规
5. **民主程序要求**：检索该制度是否涉及劳动者切身利益，是否需要履行民主程序
6. **国资监管特别要求**：如为国有企业，检索国企监管的特殊合规要求

### 0.3 输出联网学习摘要
在审查意见书的开篇，必须包含"联网学习综述"一节，简要列出：
- 检索到的最新法律法规（名称+施行日期）
- 核心监管政策要点
- 对本次制度审查的具体影响

## 一、核心法律依据

### 1.1 基础法律
- 《中华人民共和国民法典》
- 《中华人民共和国劳动合同法》（2013修正）
- 《中华人民共和国会计法》（2024年6月28日修正，**2024年7月1日施行**）
- 《中华人民共和国公司法》（2023修正，2024年7月1日施行）
- 《中华人民共和国招标投标法》
- 《中华人民共和国安全生产法》（2021修正）
- 《中华人民共和国消防法》

### 1.2 监管规定
- 《企业内部控制基本规范》（财政部，财会〔2008〕7号）
- 《企业内部控制应用指引》第8号——资产管理
- 《企业内部控制应用指引》第15号——全面预算
- 《中央企业合规管理办法》（国务院国资委令第42号，2022年10月施行）
- 《山西省省属企业合规管理办法（试行）》（晋国资发〔2023〕10号）
- 国资委党委《关于建立和完善中央企业职工代表大会制度的指导意见》（国资党委群工〔2007〕120号）
- 《企业民主管理规定》（总工发〔2012〕12号）

### 1.3 审查基本原则
1. **合法性审查**：制度内容不得违反法律、行政法规的强制性规定
2. **合规性审查**：是否符合国资监管机构的专项要求
3. **程序性审查**：制定程序是否合规（民主程序、公示程序）
4. **可执行性审查**：条款是否具有可操作性，能否在实际业务中落实
5. **逻辑一致性审查**：制度内部条款之间、制度与上位制度之间是否矛盾
6. **风险控制审查**：是否存在管理漏洞或重大风险敞口

## 二、制度审查工作流（关键——必须按此顺序执行）

### 第一步：背景调查（审查前准备，15-20%时间）
1. **读取工作区全部文件**：进入制度所在的工作区路径，列出并阅读所有文件（含原稿件、格式模板、背景资料）
2. **识别企业性质**：是否为国有企业/国有独资/国有控股/国有参股——直接影响审查标准和合规要求
3. **关联制度梳理**：企业内部是否存在关联制度（如存货盘点制度可能关联采购制度、仓储制度、财务制度）
4. **关联集团制度**：该企业所属集团是否有相关统一制度，本制度是否为对集团制度的细化/补充

### 第二步：粗审（15-20%时间）
- 通读全文，了解制度结构和核心规范内容
- 识别制度类型（管理办法/实施细则/操作规程/管理规定等）
- 判断制度的完整性和体系性
- 定位高危条款区域

### 第三步：精审（50-60%时间）
- 逐条审查，从以下维度深入分析每一条款：
  - **合法性维度**：是否违反法律强制性规定
  - **合规性维度**：是否符合国资监管要求
  - **逻辑性维度**：表述是否清晰、无歧义，上下文是否一致
  - **可操作性维度**：在实际工作中能否落地执行
  - **完整性维度**：是否存在应当规定而未规定的内容

### 第四步：复核（10-15%时间）
- 检查制度内部条款之间是否存在矛盾和冲突
- 检查制度名称与内容的匹配度
- 检查制度与上位法律、集团制度的衔接
- 复核高风险条款

### 第五步：出具意见
- 撰写审查意见书（.docx 格式），使用 python-docx 生成 Word 文档
- 审查意见书结构见第三节

## 三、审查意见书结构

### 3.1 审查意见书（自由格式版）结构
1. **审查结论**（通过/附条件通过/退回重大修改）
2. **联网学习综述**（法规政策检索结果）
3. **整体评价**（制度体系完整性、主要亮点和不足）
4. **重大风险提示**（必须修改——标注"致命"/"高危"等级）
5. **一般修改建议**（建议修改）
6. **提示性建议**（可改可不改、缺失条款补充）

### 3.2 制度审查的专项审查维度

#### A. 程序合规审查（重要）
- [ ] 制度是否经过民主程序（如涉及劳动者切身利益）
- [ ] 制度是否经过公示告知程序
- [ ] 制度是否经过企业内部审批程序
- [ ] **集团制度适用问题**：本企业直接适用集团制度时，是否已按照本企业名义履行了本企业的法定程序

#### B. 内容合法性审查
- [ ] 是否存在违反法律强制性规定的条款
- [ ] 是否存在超越企业权限的条款
- [ ] 处分措施是否合法（罚款是否违法、扣款是否有法律依据）

#### C. 内容可操作性审查
- [ ] 时间节点是否合理可行（如当月盘点次日报送是否现实）
- [ ] 职责分工是否明确、无重叠
- [ ] 工作流程是否完整闭环
- [ ] 异常情况是否有处理规则
- [ ] 责任追究是否具体可执行

#### D. 逻辑一致性审查
- [ ] 制度名称与内容是否一致
- [ ] 制度内部的术语定义是否统一
- [ ] 各章节条款之间是否自洽
- [ ] 制度与前版制度、上位制度是否衔接

## 四、常见制度审查要点清单

### 4.1 程序类问题
1. **民主程序缺失**：制度涉及劳动者切身利益的（劳动报酬、工作时间、休息休假、劳动安全卫生、保险福利、职工培训、劳动纪律、劳动定额管理等），必须经职工代表大会或全体职工讨论，与工会或职工代表平等协商确定
2. **公示告知程序缺失**：制度须向全体员工公示（公告栏、OA系统、邮件送达、员工手册签收等）
3. **制度层级管理**：不同层级制度（公司章程→董事会决议→基本制度→管理办法→实施细则）的效力层级和衔接

### 4.2 国有企业的特殊合规要求
1. **国资监管红线**：国有资产流失防范、关联交易管理、"三重一大"决策程序
2. **集团制度适用**：多级法人治理结构下，子公司适用集团制度时需独立履行民主程序和公示程序
3. **合规管理要求**：《山西省省属企业合规管理办法（试行）》（晋国资发〔2023〕10号）要求将合规要求嵌入各管理环节
4. **"管业务必须管合规"**：各业务部门是合规管理的第一道防线

### 4.3 可操作性常见问题
1. **时间节点不合理**：如规定的时间过短，无法在实际业务中完成
2. **职责分工模糊**：使用"相关部门""有关人员"等模糊表述
3. **责任追究虚化**：只说"追究责任"但不明确谁来追究、追究程序、追究标准
4. **操作流程不完整**：只规定正常流程，缺乏异常情况处理机制
5. **表单名称不统一**：制度中引用的表格名称与实际不符或未规范命名

### 4.4 逻辑一致性问题
1. **定义不一致**：制度名称、关键术语在全文中的使用不一致
2. **范围矛盾**：总则中的适用范围和具体条款中的适用范围不一致
3. **前后条款冲突**：不同章节对同一事项的规定相互抵触

**发现的关键问题模式（已提炼为综合审查要点，供后续审查复用）**：
1. **制度名称与文件名不一致**——文实不符，是常见的基础性瑕疵
2. **缺乏上位法依据**——很多企业制度只说"结合公司实际"而不列明法定依据
3. **盘点范围与准备资料范围矛盾**——总则说A但流程中要求B，前后脱节

6. **表单名称不规范**——XX占位符未替换
7. **责任追究虚化**——只说"追究责任"但不明确程序标准
8. **监督责任与人力资源不匹配**——"全程监督"在实际中难以落实
9. **共同解释权无兜底机制**——两个部门共同解释但缺失冲突裁断
10. **民主程序/公示程序提示**——劳动纪律类制度须注意此问题

**新增审查要点**（§4.2-4.4）：在可操作审查和逻辑一致性审查条目中均已覆盖上述发现。
