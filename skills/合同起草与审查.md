## 功能说明

本 Skill 用于处理**合同起草与审查**相关的法律事务，涵盖合同起草、条款设计、合同审查、风险评估、修改建议等全流程服务。

**核心知识体系**：本技能整合了"中国合同标准"系列四部专著的知识精华：
1. 《合同起草审查指南：三观四步法》（第五版，2024）——何力、常金光等
2. 《合同起草审查指南：公司及并购卷》（2024）——何力、常金光等
3. 《合同审查思维体系与实务技能》（第二版，2020）——张海燕
4. 《合同起草审查指南：房地产开发卷、建设工程卷》（2023）——鞠美园、何力等

### 0.1 识别合同涉及的行业/领域
根据合同标题、标的和内容，提取关键词（如：低空经济、文物管护、建设工程、医疗合规、数据安全、房地产开发、公司并购等）。

### 0.2 联网检索（至少3个方向）
根据合同领域，至少从以下方向中选择3个进行联网检索：
1. **法律法规动态**：检索该领域最新法规、修订动态（注意时效性——优先使用2025-2026年最新版本）
2. **监管政策**：检索行业主管部门的最新政策文件、规范要求
3. **地方规定**：如合同涉及特定省份/城市，检索当地的地方性法规和执行标准
4. **判例参考**：检索相关纠纷的裁判案例，了解法院的裁判倾向和关键风险点
5. **权威解读**：检索主管部门的政策解读、权威法律评论文章

## 一、核心方法论："三观四步法"知识体系

### 1.1 知识体系总览

"三观四步法"是本 Skill 的核心方法论，源自《合同起草审查指南：三观四步法》。

**"三观四步法"= "三观"知识体系 + "四步"操作流程 + "理念"指导思想**

```
                    ┌─────────────────────────────┐
                    │     理念（贯穿始终）          │
                    │  指引宏观、中观、微观        │
                    └─────────────────────────────┘
                                    │
            ┌───────────────────────┼───────────────────────┐
            ▼                       ▼                       ▼
    ┌───────────────┐     ┌───────────────┐     ┌───────────────┐
    │  宏观          │     │  中观          │     │  微观          │
    │  交易结构      │     │  合同形式      │     │  合同条款+语言  │
    │  什么交易？    │     │  什么文本？    │     │  什么条款？    │
    └───────────────┘     └───────────────┘     └───────────────┘
```

**"三观"的三个层面**：
- **宏观——交易结构**：这个交易能不能做、风险大不大？采取什么交易结构做？
- **中观——合同形式**：采取什么合同形式？是简单合同还是复杂成套合同？
- **微观——合同条款+合同语言**：合同条款要怎么修订？语言是否精确？

### 1.2 "理念"层面——合同起草审查的指导思想

在"三观"之前和之上，有"理念"贯穿整个合同工作：

1. **促进交易与防范风险的平衡**：合同审查的两个核心目的是防范风险和促进交易，二者不可偏废
2. **从己方立场出发**：以"合同签订后权利义务关系明确、履行顺畅、违约有后果、任何一种可能下己方风险可控"为最终目标
3. **全面思考与重点突出**：既要全面梳理合同权利义务体系，又要抓住关键风险点
4. **理性决策与商业判断**：律师识别并分析风险，但不能代替委托人做商业决策

### 1.3 "三观"知识体系详解

#### 宏观——交易结构

**核心问题**：这个交易能不能做？采取什么交易结构？

**交易结构分析要素**：
1. **合同类型**：买卖合同/服务合同/租赁合同/合作协议/投资协议等，注意合同名实是否一致
2. **合同主体**：谁和谁交易？主体是否适格？是否有资质要求？是否存在表见代理风险？
3. **合同标的**：交易什么？标的是否清晰？权属来源是否合法完整？
4. **合同程序**：是否需要审批/备案/招投标/股东会决议/进场交易等前置程序？
5. **交易结构设计**：是资产收购还是股权收购？是买卖还是合作？是委托还是服务？

**基础交易结构类型**（源自公司及并购卷）：
- 资产收购型：直接购买资产的所有权或使用权
- 股权收购型：通过取得股权间接控制目标资产
- 复合型交易结构：同时涉及多种交易安排
- 资金回报型（名股实债）：表面股权投资实为借贷关系
- 广义协议控制：通过协议安排实现控制而非股权

#### 中观——合同形式

**核心问题**：采取什么合同形式？

**合同形式的分类与选择**：
1. **简单合同 vs 复杂成套合同**：
   - 简单合同：单一文本，适用于简单交易
   - 复杂成套合同：主交易合同+附件+配套协议，适用于复杂交易（如并购、房地产开发）
2. **合同文本层次结构**（"三层文本结构"）：
   - 第一层：主交易合同（核心交易安排）
   - 第二层：主交易合同的附件（补充约定、技术标准、清单等）
   - 第三层：配套协议（保密协议、服务协议、担保协议等）
3. **意向书/备忘录/框架协议**：区分"预约定"效力与"本约"效力
4. **格式条款与格式合同**：注意《民法典》第496-498条的规制
5. **合同与附件的关系**：明确效力层级和冲突处理规则

**交易前期文本与履行中文件**（并购场景）：
- 前期文本：意向书、条款清单(Term Sheet)、保密协议、尽职调查清单
- 正式交易文本：股权转让协议/增资协议+章程修正案+配套协议
- 履行中文件：交割确认书、交接清单、通知函

#### 微观——合同条款

**核心问题**：合同条款要怎么修订？

**条款起草审查的系统方法**（源自三观四步法）：
1. **"三点一线法"**：
   - 起点：从商业目的出发确定核心权利义务
   - 中线：沿交易流程设计条款顺序和逻辑
   - 终点：确保争议解决和救济措施的有效性
2. **一般条款知识**：对通用条款（违约责任、管辖、保密等）的标准化认知
3. **典型条款与配套条款**：区分核心交易条款和辅助保障条款

#### 微观——合同语言

**核心要求**：精确、无歧义、可操作。

1. **精确化**：杜绝"合理期限""适当赔偿""重大违约"等模糊用语
2. **量化**：期限、金额、比例、标准等必须可量化
3. **一致性**：同一概念使用同一术语，前后表述一致
4. **完整性**：条件、例外、后果等表述完整

### 1.4 "四步工作流程"

**第一步：背景调查（审查前准备，15-20%时间）**
1. 读取工作区全部背景资料
2. 梳理关联合同链条（如涉及土地、资产、前手交易）
3. 识别合同主体真实地位（比对合同陈述与背景材料）
4. 检索行业监管政策（联网学习）
5. 确认地方执行标准
6. 评估关联合同影响

**第二步：三观分析（最关键步骤，50-60%时间）**
1. **宏观分析**：判断交易结构是否合理、合法、风险可控
2. **中观分析**：判断合同形式是否适当、文本结构是否完整
3. **微观分析**：逐条审查合同条款和语言的精确性

**第三步：复核（10-15%时间）**
- 重点复核高危条款（红旗条款）
- 检查前后条款是否一致
- 检查主合同与附件的效力层级和条款一致性

**第四步：出具意见**
- 撰写审查意见书（.docx格式）
- 创建带修订标记的合同副本（.docx格式）
- 使用 python-docx 生成 Word 文档

## 二、合同审查思维体系（源自张海燕《合同审查思维体系与实务技能》）

### 2.1 合同审查的两个核心目的
1. **防范风险**：识别并控制合同中可能存在的法律风险
   - 识别风险（专业型风险 + 经验型风险）
   - 分析风险（法律风险、技术风险、信用风险）
   - 提出防范措施建议
2. **促进交易**：在控制风险的前提下推动交易达成

### 2.2 合同审查的三个基本前提
1. **修改立场：代表谁**——必须明确审查立场（甲方/乙方/中立）
2. **修改尺度：谈判地位**——根据委托人的谈判地位决定审查强度
3. **修改模式：以客户为中心**——根据客户的商业目的和风险承受能力定制方案

### 2.3 合同审查的四个步骤
1. **审查交易模式**：判断交易模式的合法性、可行性和风险
2. **审查合同结构**：检查合同文本结构的完整性
3. **审查合同条款**：逐条审查条款的合法性和周延性
4. **审查文字符号**：检查文字表述、标点符号、数字的准确性

### 2.4 合同审查的五个标准
1. **合规性**：不违反法律、行政法规的强制性规定
2. **周延性**：条款覆盖各种可能情形，无重大遗漏
3. **可操作性**：条款约定可实际执行
4. **公平性**：权利义务基本对等
5. **倾向性**：在合法前提下适度向委托人倾斜

### 2.5 提升合同审查水平的六个途径
1. **多想（动脑）**：深入思考交易背景和商业逻辑
2. **多看（动眼）**：广泛阅读各类合同和判例
3. **多问（动嘴）**：与客户充分沟通，了解交易细节
4. **多练（动手）**：大量实践，积累经验
5. **多学（动智）**：持续学习新法规、新知识
6. **多点责任心（动心）**：以高度责任心对待每一份合同

### 2.6 合同审查的常用方法

| 方法 | 适用场景 | 操作要点 |
|------|----------|----------|
| **补增** | 缺失必要条款或约定不完整 | 补充缺失的条款或内容，使合同完整周延 |
| **删改** | 存在违法条款、矛盾条款或对己方不利条款 | 删除或修改不当内容，替换为合法有利条款 |
| **建议** | 存在可优化但非必须修改之处 | 提出优化建议，由委托人决定是否采纳 |
| **提示** | 存在无法修改但需委托人知晓的风险 | 明确提示风险，让委托人心中有数 |
| **综合运用** | 复杂合同 | 根据条款性质灵活组合使用上述方法 |

### 3.3 审查基本原则
1. **合法性审查**：合同内容不得违反法律、行政法规的效力性强制性规定，不得违背公序良俗（民法典第153条）
2. **完整性审查**：合同要素是否齐全，是否存在重大遗漏
3. **明确性审查**：条款表述是否清晰、无歧义，权利义务是否对等明确
4. **可执行性审查**：约定是否具有可操作性，违约责任是否切实可行
5. **风险控制审查**：是否存在重大法律或商业风险敞口

## 四、合同起草与审查工作流（关键——必须按此顺序执行）

### 第一步：背景调查（审查前准备，15-20%时间）
**在开始逐条审查合同条款之前，必须先完成以下工作**：

1. **读取工作区全部背景资料**：进入合同所在的工作区路径，列出并阅读所有文件
2. **梳理关联合同链条**：如涉及土地、资产、前手交易，须追溯权属来源（如土地承租→转租→合作开发的链条），逐一查阅关联合同
3. **识别合同主体真实地位**：比对合同中的陈述与背景材料，发现主体资格、权属来源等方面的虚假/不准确陈述
4. **检索行业监管政策**：针对合同涉及的行业领域（如低空经济、建设工程、医疗等），主动联网检索最新法规政策动态
5. **确认地方执行标准**：如涉及特定省份，检索该省的地方性规定和裁量标准
6. **评估关联合同影响**：分析其他合同对本合同的约束和影响（如原租约对转租/合作/用途变更的限制）

### 第二步：粗审（三观分析，15-20%时间）
- 通读全文，了解合同结构和核心交易安排
- 运用"三观分析法"进行初步判断：
  - **宏观**：交易结构是否合理？
  - **中观**：合同形式是否适当？
  - **微观**：关键条款是否存在重大缺陷？
- 识别合同性质（买卖合同/服务合同/租赁合同/合作协议/框架协议等）
- 判断合同名实是否一致（如名为"合作方案"实为正式合同）
- 定位高危条款区域

### 第三步：精审（50-60%时间）
- 逐条审查，标注风险点，查阅法律依据
- **从己方立场出发**，逐一检视每一条款对己方的影响
- 重点审查：权利义务对等性、违约责任、解除条件、争议解决
- 审视每一种可能场景下己方的风险敞口
- 运用"补增、删改、建议、提示"方法提出修订意见

### 第四步：复核（10-15%时间）
- 重点复核高危条款（红旗条款）
- 检查前后条款是否一致、是否有矛盾
- 检查主合同与附件的效力层级和条款一致性

### 第五步：出具意见
- 撰写审查意见书（.docx 格式）
- 创建带修订标记的合同副本（.docx 格式，与原合同排版一致）
- 使用 python-docx 生成 Word 文档（红色删除线=删除、蓝色下划线=新增、橙色批注=修改原因与依据）
- 审查意见书和修订对照稿可合并为一份 .docx，也可分开输出为两份 .docx

## 五、用户工作习惯与偏好（必须遵守）

### 5.1 审查立场
- 用户会明确指定审查立场（甲方/乙方/中立），必须在审查意见中明确标注审查立场
- 从指定立场出发，以"**合同签订后权利义务关系明确、履行顺畅、违约有后果、任何一种可能下己方风险可控**"为最终审查目标

### 5.2 审查关注重点（按优先级排序）
1. **主体资格与权属**：合同主体是否适格、权属来源是否合法完整——这是合同的根基
2. **合规审批**：涉及特殊行业的资质许可、用途变更审批、空域/环境/土地等监管审批
3. **收益/对价保障**：己方的核心利益是否有切实保障机制（保底条款、核查权、违约金）
4. **违约责任**：是否对等、是否具体可执行、是否有责任上限
5. **争议解决**：管辖是否对己方有利、仲裁/诉讼选择是否明确有效
6. **合同期限与退出机制**：期限是否合理、解除条件是否明确、解除后处理是否清晰

### 5.3 工作区要求
- 每次合同审查，必须首先进入工作区路径读取全部背景资料
- 关注工作区内其他相关合同的关联性与影响
- 审查结果输出到原工作区目录内

### 5.4 输出格式要求

#### 输出文件
- **审查意见书**（.docx Word格式）：一份完整的审查意见书，包含下文所述全部结构
- **修订对照稿**（.docx Word格式）：带修订标记的合同副本，与原合同格式一致

> 审查意见书也可与修订对照稿合并在同一份 .docx 文件中（前半部分审查意见，后半部分修订对照稿），视文件长度和可读性决定。

#### 文件格式规范
- 输出格式为 **.docx**（Word文档），不得输出 .md 格式
- 原稿为 .doc 格式的，先通过 textutil 转换为文本读取内容，输出时生成 .docx 格式
- 修订对照稿应尽量还原原合同的排版结构（标题层级、条款编号、段落格式）
- 文件命名：在原文件名前加"【审查稿】"前缀，扩展名改为 .docx

#### 审查意见书结构
1. 审查结论（通过/附条件通过/退回重大修改）
2. 联网学习综述（法规政策+判例检索结果）
3. 背景分析与关联合同梳理（如有）
4. 重大风险提示（必须修改——每条标注"致命"/"高危"等级）
5. 一般修改建议（建议修改）
6. 提示性建议（可改可不改、缺失条款补充）
7. 审查总结（前置条件清单 + 底线判断）

#### Word文档中的修订标注规范

| 修订类型 | Word呈现方式 | python-docx 实现 |
|----------|-------------|------------------|
| **删除内容** | 红色删除线文本 | `run.font.color.rgb = RGBColor(0xFF, 0x00, 0x00)` + `run.font.strike = True` |
| **新增内容** | 蓝色下划线文本 | `run.font.color.rgb = RGBColor(0x00, 0x00, 0xFF)` + `run.font.underline = True` |
| **批注说明** | `[批注：修改原因与法律依据]` | 橙色括号文本紧随修改之处 |
| **原条款（保留对照）** | 灰色文本 | `run.font.color.rgb = RGBColor(0x80, 0x80, 0x80)` |

每条修订必须附 `[批注：...]` 说明修改原因和法律依据。

#### 生成 .docx 的技术要求
1. 使用 `python-docx` 库创建文档
2. 设置页面为标准A4纸，页边距上下2.54cm、左右3.18cm
3. 正文使用宋体/仿宋 12pt，标题使用黑体加粗
4. 条款编号应保留原合同的编号体系
5. 原条款（待删除内容）用红色删除线标识
6. 建议新增内容用蓝色下划线标识
7. 每处修改后紧跟批注，批注用橙色小号字体（9pt）缩进显示

### 5.5 特定客户的格式要求——{{公司}}

**对于{{公司}}的所有合同法审**，必须按照该公司法审意见书表格格式输出：

#### 审查意见书结构（辰润公司专用）
1. **审查结论**（附条件通过/退回重大修改）
2. **基本情况说明**（表格）：送审编号、合同名称、合同相对方、送审人、送审组织、送审时间、合同金额、履行期限、建议审查目的方向和主要内容
3. **法审文件审查意见**（三栏表格）：条款号 | 修改后内容或律师建议 | 涉及法律风险
4. **相关合同审查意见**（三栏表格）：条款号 | 修改后内容或律师建议 | 涉及法律风险
5. 审查律师签字、律师事务所盖章、审查时间

#### 表格格式规范（辰润公司专用）

- 正文使用宋体10pt-10.5pt
- 风险等级标注（致命/高危/一般）
- 每个辰润合同审查需输出**两份文件**：详细版+填表用版

### 5.6 行文风格
- 审查意见应直接、务实，不做学理探讨式的冗长分析
- 风险定性明确（致命/高危/一般/提示），不模糊
- 修改建议须具体可操作，给出替代条款文本
- 引用法条时注明具体条、款、项

## 六、逐条审查清单

### 6.1 形式审查
- [ ] 合同名称与内容是否一致
- [ ] 当事人信息是否完整准确（名称、统一社会信用代码、法定代表人、地址、联系方式）
- [ ] 签署日期、地点是否填写
- [ ] 附件是否齐全
- [ ] 签署页是否完整

### 6.2 实质审查（逐条过审）

#### 标的条款
- 标的物/标的权利描述是否清晰、具体、唯一
- 数量、质量标准是否有明确约定
- **对于土地/房屋类标的**：必须追溯权属链条，核实签约方的权利来源
- **对于股权类标的**：核实出资是否实缴、是否存在权利负担

#### 价款/报酬/收益分配条款
- 金额大小写是否一致
- 计价货币、税率、含税/不含税
- 支付方式、支付节点、分期安排
- 发票类型及开具时间
- **对于收益分配类**：核算口径是否明确、己方是否有核查权、是否有保底机制

#### 期限条款
- 合同生效条件与生效日期
- 履行期限、各阶段时间节点
- 续期机制（自动续期/协商续期）
- **对于涉及前手权利的期限**：不得超过前手权利剩余期限

#### 权利义务条款
- 双方权利义务是否对等
- 是否存在单方加重对方责任的格式条款（民法典第497条）
- 通知送达条款是否完善
- **信息不对称场景下**：己方的知情权、查阅权、监督权是否充分

#### 保密条款
- 保密信息的范围和例外
- 保密期限（通常延续至合同终止后2-5年）
- 违约责任

#### 知识产权条款
- 原有知识产权归属
- 履行过程中产生的知识产权归属
- 授权范围

#### 违约责任（重点关注）
- 违约情形是否全面覆盖
- 违约金比例是否合理（民法典第585条，违约金过高可请求调减）
- 损失赔偿范围是否明确（直接损失、间接损失；是否设上限）
- 定金条款（定金罚则，民法典第587条，定金不超过主合同标的额20%）
- **违约金是否具体可量化**：杜绝"合理标准""相应赔偿"等模糊用语
- **双方违约责任是否对等**：尤其关注违约金比例和触发条件

#### 免责条款（不可抗力）
- 不可抗力定义范围
- 通知义务和减损义务
- 不可抗力事件发生后的处理方式（顺延/解除/费用承担）

#### 争议解决条款（关键审查项）
- **仲裁 vs 诉讼**的选择明确
  - 约定仲裁：需明确仲裁机构名称（一字不差）
  - 约定诉讼：需明确管辖法院，不得违反专属管辖和级别管辖
  - 禁止同时约定仲裁和诉讼（仲裁协议无效风险）
- 管辖地选择对己方是否有利
- **主合同与附件管辖约定是否一致**

#### 合同解除条款（重点关注）
- 约定解除条件是否具体量化（避免"严重违约""重大损失"等模糊用语）
- 解除后的处理（返还、赔偿、清算）是否明确
- 解除权行使期限
- **己方退出机制**：是否在任何一种可能场景下都能安全退出

#### 合同生效条件
- 是否设定了合理的生效先决条件
- 先决条件是否可验证、有明确的时间节点
- 条件不满足时的处理方式

#### 签署条款
- 签字盖章是否具有同等效力
- 盖章类型要求（公章/合同专用章）
- 是否所有必要签约方都已列入签署页

## 七、风险点速查

### 7.1 高危条款（红旗条款）
- **主体资格虚假陈述**：签约方声称的权利与实际不符——合同基础不成立，可被撤销（民法典第148条）
- **格式条款**：提供方是否履行提示说明义务，是否排除对方主要权利
- **违约金过高**：超过实际损失30%的违约金可能被调减
- **管辖条款**：明显加重对方维权成本的管辖约定
- **空白签署**：先签字后补填内容的合同
- **概括性授权**：授权范围过于宽泛
- **无限责任**：未设责任上限的赔偿条款
- **隐藏义务**：小字或附件中隐含的义务
- **权属链断裂**：签约方的前手权利存在瑕疵或限制
- **检测报告质量保证缺失**（技术服务类致命风险）
- **全额预付**（失去付款控制权的高危风险）
- **"起诉方所在地法院"管辖**（形式公平实质不利）
- **名实不符合同**（名为合作实为转让、名为投资实为借贷等）
- **阴阳合同**：备案合同与执行合同不一致

### 7.2 常见陷阱
- **口头补充**：以口头约定变更书面合同
- **倒签合同**：合同签署日期早于实际签署日期（审计合规风险）
- **关联交易**：未披露的关联方交易
- **抽屉协议**：签署未注明日期的补充协议
- **框架协议陷阱**：名为"方案""备忘录""意向书"，但实际产生约束力的条款已在执行
- **"按民法典相关规定承担违约责任"**：虚置条款，相当于未约定任何具体违约责任
- **"费用另行协商确定"**：实质上无约束力
- **概括性约定续期条件**：给对方无限续期权

## 八、常见合同类型起草与审查要点

### 8.1 买卖合同
- 标的物所有权转移时间节点
- 风险转移（交付前卖方承担/交付后买方承担，民法典第604条）
- 验收期限和验收标准
- 质量异议期和质保期
- 包装、运输、保险费用承担

### 8.2 服务合同/委托合同
- 委托事项范围是否明确
- 转委托是否需经委托人同意
- 费用承担和报酬支付
- 任意解除权及损失的赔偿范围（民法典第933条）
- **注意区分服务合同与委托合同的法律适用差异**

### 8.3 租赁合同
- 租赁物描述和用途限制
- 租赁期限不得超过20年（民法典第705条）
- 装修/改造的审批和归属
- 优先承租权和优先购买权
- **转租须经出租人同意**（民法典第716条）

### 8.4 建设工程合同

#### 宏观——交易结构审查要点
1. **合同类型识别**：建设工程合同 vs 非建设工程合同，注意承揽合同与建设工程合同的区分
2. **合同主体资质**：施工总承包/专业承包/劳务分包等资质等级要求
3. **违法情形识别**：
   - 非法转包（全部工程转给第三方）
   - 违法分包（分包给无资质方/未经发包人同意）
   - 挂靠承包（借用资质）
   - 支解发包（将应由一个承包人完成的工程肢解发包）
4. **招投标合规**：必须招标项目范围、招标程序合规、"阴阳合同"处理
5. **合同程序**：建设工程规划许可、施工许可等前置审批
6. **工程合同的违法无效与应对**：违法合同无效后的工程款结算和保护

#### 中观——合同形式审查要点
1. **施工合同示范文本**（2017版）：通用合同条款+专用合同条款+协议书
2. **复杂成套合同结构**：施工合同+监理合同+分包合同+担保合同+质量标准附件
3. **EPC工程总承包合同**的特殊文本结构

#### 微观——合同条款审查要点（基于2017版施工合同）
1. **工程量+工程价款条款**：计价方式（固定总价/固定单价/成本加酬金）、工程量清单、变更估价
2. **工期条款**：开工日期、竣工日期、工期顺延条件
3. **质量条款**：质量标准、验收程序、质量保证金
4. **违约责任**：工期延误违约金、质量不合格违约金
5. **默示条款**：施工合同中发包人逾期不回复视为认可的默示条款风险
6. **"霸王条款"**：发包人利用优势地位设置的不公平条款
7. **优先受偿权**：《民法典》第807条的建设工程价款优先受偿权
8. **垫资承包与利息约定**：垫资的合法性及利息约定

#### 特殊合同类型
- **内部承包合同**：合法性判断（区别于挂靠承包）、风险防控
- **劳务分包合同**："包工头"劳务承包的风险
- **专业分包合同**：指定分包、独立发包的特殊情形
- **建设工程勘察、设计合同**
- **建设工程监理合同**

### 8.5 技术开发/技术服务合同
- 技术成果归属（委托开发/合作开发）
- 后续改进的权利归属
- 技术风险的承担
- **检测类**：资质预审（CMA/CNAS）、报告质量保证、付款分阶段、异议复检程序

### 8.6 合资/合作协议
- 出资方式和出资期限
- 决策机制和表决权
- 利润分配和亏损分担
- 退出机制（转让/回购/解散）
- 僵局处理机制
- **信息不对称场景下的知情权和核查权**（尤为重要）
- **土地合作类**：必须审查土地权属链条的完整性和合法性

## 九、公司及并购类合同专项知识（源自《公司及并购卷》）

### 9.1 "公司及并购类合同"整体认识框架

公司及并购类合同分为两大类：
1. **公司类合同**：出资协议、章程、增资协议、股权转让协议、股东协议等
2. **并购类合同**：资产收购、股权收购、合并、分立等

**分析框架**：运用"三观分析法"——先分析宏观交易结构（主交易结构+配套安排），再分析中观合同形式，最后分析微观合同条款。

### 9.2 宏观——主交易结构类型

#### 基础交易结构
- **资产收购型**：直接购买目标资产（土地使用权、房屋、机器设备、知识产权等）
- **股权收购型**：通过取得目标公司股权间接控制目标资产
- **合并/分立型**：公司合并（吸收合并/新设合并）、公司分立

#### 复合型交易结构
- **股权收购+资产重组**：先股权收购，后内部资产整合
- **增资+股权转让**：混合方式取得控制权
- **资产收购+业务合作**：购买资产同时建立合作关系

#### 资金回报型交易结构（名股实债）
- 形式上为股权投资，实质上为借贷关系
- 典型特征：固定收益回报、回购条款、不参与经营管理
- **审查重点**：名股实债的法律效力认定、回购安排的可行性

#### 特殊交易结构——广义协议控制
- 通过一系列协议安排实现对目标主体的实际控制
- 常见场景：VIE架构、委托投票权、一致行动协议

### 9.3 宏观——配套安排

1. **公司/股东利益输送安排**：关联交易、利润分配政策、服务费安排
2. **股权权能（身份权）安排**：表决权调整、一票否决权、董事会席位安排
3. **主交易相关债务处理（承债式收购）**：
   - 债务类型识别（金融债务/经营债务/或有债务）
   - 典型债务处理方式（由转让方清偿/由受让方承接/债转股等）
   - 债权人配合的必要性
4. **投资人保护安排**：对赌协议、反稀释、优先清算权、随售权/拖售权
5. **基本限制**：股东、董事、监事、高管对公司的信义义务（忠实义务+勤勉义务）

### 9.4 中观——并购类合同的文本结构

#### 复杂收购型合同的文本结构特点
1. **"三层文本结构"**：
   - 第一层：主交易合同（股权转让协议/增资协议/资产收购协议）
   - 第二层：主交易合同的附件（清单、报表、披露函、保证函等）
   - 第三层：配套协议（保密协议、竞业禁止协议、服务协议、担保协议等）

2. **"交易前期文本"与"履行中文件"**：
   - 前期文本：意向书(LOI)、条款清单(Term Sheet)、保密协议(NDA)、尽职调查清单
   - 正式交易文本：主合同+附件+配套协议
   - 履行中文件：交割确认书、交接清单、变更登记文件

3. **公司章程、章程修正案的起草审查**：与股权交易相配套的公司治理文件

### 9.5 微观——并购类合同条款审查

#### "三点一线法"的具体运用
- **起点**：明确交易目的——取得控制权/财务投资/战略合作
- **中线**：沿交易流程设计条款——签约→交割→过渡期→整合→退出
- **终点**：确保救济措施有效——违约责任、赔偿、争议解决

#### 股权类合同核心交易条款
1. **交易价格与定价机制**：估值方法、价格调整机制（Earn-out、锁箱机制）
2. **交割先决条件**（CPs）：政府审批、第三方同意、无重大不利变化等
3. **陈述与保证条款**：转让方对目标公司状况的全面保证
4. **赔偿条款**（Indemnity）：赔偿范围、赔偿上限、赔偿期限（存活期）
5. **过渡期安排**：交割前目标公司的经营限制
6. **竞业禁止**：转让方在交易后的竞业限制

#### 资产收购类合同交易条款
1. **资产清单与范围界定**：哪些资产纳入交易、哪些排除
2. **债务处理**：随资产转移的债务和不随资产转移的债务
3. **员工安置**：人员随资产转移的方案
4. **在建工程转让**：已完成的工程量确认、后续建设责任
5. **资产交付标准**：交付时的状态、验收标准

### 9.6 特定主体的公司合同

- **股份公司**：2023版《公司法》修订后，股份公司与有限公司的差异缩小；注意股份公司类别（公众公司/非公众公司）
- **合伙企业**：普通合伙(无限责任) vs 有限合伙(有限责任)；合伙协议的核心条款
- **上市公司**：特殊监管要求、信息披露、要约收购
- **农民专业合作社**：特殊的组织结构和决策机制
- **个人独资企业、个体工商户**：出资人的无限责任

### 9.7 公司类合同：公司治理结构相关（2023版公司法更新）

**新《公司法》主要变化对合同的影响**：
- 法定代表人制度变化
- 董事会/监事会设置灵活化
- 股东出资加速到期制度
- 股东失权制度
- 简易减资制度

## 十、房地产开发类合同专项知识（源自《房地产开发卷》）

### 10.1 房地产开发合同的整体框架

房地产开发分为三个阶段，对应不同合同类型：

```
土地一级开发阶段  →  土地获取阶段  →  房地产销售/运营阶段
（拆迁、收储）      （一级/二级市场）   （买卖、物业）
```

### 10.2 土地一级开发类合同

#### 土地一级开发委托合同
- **合同主体**：政府（委托方）vs 开发企业（受托方）
- **合同类型**：行政协议还是民事合同？——需要个案判断
- **开发费用模式**："成本+利润"模式（管理费用一般≤成本的2%，利润率一般≤预计成本的8%）
- **土地开发成本归集**：8大类成本项目（土地报批税费、征地补偿费、拆迁安置补偿费、前期费用、开发整理费用、财务支出、临时看护管理费、其他费用）
- **土地出让收益分享条款**：
  - 合规性风险：与《预算法》第56条、《城市房地产管理法》第19条存在冲突
  - 司法裁判不一：有判例认定无效（如安庆金马房地产案），也有判例认定有效（如沈阳顺丰房地产案）
  - **审查建议**：避免直接约定土地出让金分成，改为固定服务费模式

#### 拆迁安置补偿协议
- **合同主体**：征收人、实施单位、被征收人
- **合同程序**：以房屋征收决定为前置条件
- **核心条款**：明确约定拆迁安置补偿方案

### 10.3 土地一级市场类合同

#### 国有土地使用权出让合同
- **合同程序**：必须经过招拍挂程序
- **合同标的**："毛地出让"不合法
- **出让合同限制"随地走"**：受让人继受出让合同中的全部义务和限制

#### 勾地协议（用地预申请协议）
- 土地出让前的意向性协议
- 注意区分预约与本约的效力

#### 土地置换协议
- 适用场景：政府因规划调整需要置换土地使用权

### 10.4 土地二级市场类合同

#### 国有土地使用权转让合同
- **转让限制**：
  - 已支付全部土地出让金+取得土地使用权证书
  - 完成开发投资总额的25%以上（房屋建设工程）
  - 形成工业用地或其他建设用地条件（成片开发）
- **违反限制的法律效果**：转让合同一般仍有效（管理性规范），但无法办理产权变更登记
- **"闲置土地"风险**：满一年征缴闲置费（≤出让金20%），满两年无偿收回
- **垫资开发**：为满足25%投资要求，由受让方垫资完成开发
  - 垫资开发的合同结构：借款+委托代建+主转让合同
  - 是复杂成套合同，非简单付款时间调整
- **国有资产进场交易**：涉及国有资产的须在产权交易机构公开进行
- **划拨土地使用权**：不得直接转让，须先补办出让手续

#### 房地产项目股权取得类合同
- **"合作开发模式"**的股权取得：双方以项目公司为平台合作
- **"项目转让模式"**的股权取得：直接收购项目公司股权
- **"前期处理"**：在股权取得前需完成土地使用权转让给项目公司等前期步骤
- **股权转让 vs 土地使用权转让的选择**：税负差异（股权转让可避免土地增值税）

#### 联建合同（狭义的房地产合作开发合同）
- **核心条款**：各方投入（测算、比例、时点、融资安排）、利润/房屋分配
- **报批报建手续**：由哪方负责、以谁为主体
- **房屋销售管理**：销售回款监管
- **退出清理**：物业分割、产权过户、售后责任承担
- **法律依据**：《国有土地使用权合同纠纷司法解释》第12-20条

#### 房地产委托代建合同
- **三种模式的比较**：
  | 方面 | 委托代建 | 工程总承包(EPC) | 全过程工程咨询 |
  |------|----------|----------------|---------------|
  | 法律关系 | 特殊委托合同 | 建设工程合同 | 咨询服务合同 |
  | 工作范围 | "包项目" | "包工程" | "包咨询" |
  | 风险负担 | 管理风险 | 全部工程风险 | 有限咨询责任 |
  | 对外签约 | 可以委托人名义 | 自行签约 | 一般不能代表 |
- **商业代建** vs **政府代建**模式的区别
- **代建费用**：商业代建按管理服务费，政府代建按代建管理费

#### "一级市场联合拿地"类合同
- 联合体各方通过招拍挂共同取得土地使用权
- 核心：联合体内部的权利义务分配、项目公司设立安排

### 10.5 房地产销售类合同

#### 一般商品房买卖合同
- **合同形式**：使用政府示范文本+补充协议+附件+配套协议的"复杂成套合同"
- **格式条款风险**：
  - 使用示范文本不直接等同于格式合同
  - 补充协议可能被认定为格式条款（需加粗加黑、合理提示）
  - 法院可能不采纳未经合理提示的限制性条款
- **房屋认购书（认购协议）**：
  - 预约 vs 本约的区分
  - 违约责任条款清晰化可避免本约/预约争议
  - 需明确签订本约的条件和时间
- **委托装修协议**：
  - 建议由买受人与第三方装修公司直接签署
  - 避免开发商承担连带责任
  - 注意房价限价政策下将部分房款作为装修款的合规风险
- **售后返租/售后包租**：商业地产常用模式，注意返租年限和回报承诺的可行性

#### 特殊房屋销售类合同
- 经济适用房、限价商品房、共有产权住房等特殊类型

#### 物业服务合同
- 前期物业服务合同（开发商代为签订）
- 物业服务合同的示范文本

### 10.6 专题：特殊场景下的房地产开发合同

#### 集体建设用地类合同
- 宅基地的交易模式限制
- 集体经营性建设用地的出让、出租

#### 城市更新项目类合同
- 旧村改造（合作改造）模式的合同文本
- 城市更新中多主体合作的合同安排

## 十一、特定合同类型审查要点（从原Skill继承并升级）

### 11.1 土地经营权流转/合作合同
- 土地权属来源（所有权→承包权→经营权/承租权链条）
- 土地性质（耕地/林地/建设用地，是否涉及永久基本农田）
- 用途变更是否需要审批（《土地管理法》第44条）
- 原租约对转租/再流转/用途变更的限制
- 集体土地民主决议程序（"四议两公开"）
- 地上建筑物/构筑物的权属和处置
- 合作期限不得超过前手权利的剩余期限

### 11.2 文物管护使用协议
- **甲方权利来源验证**：甲方对文物的管理使用权来源（所有权人？认养人？代管人？），须在鉴于条款中说明，权利来源文件作为附件
- **主体空白**：甲方身份空白是本类合同的致命缺陷
- **三重监管合规**：文物局+民宗局+消防救援——三方监管要求必须在合同中体现
- **宗教团体使用寺庙的合规性**：须事先取得文物和宗教两部门书面意见
- **保险条款**：必须要求使用人购买公众责任险+文物建筑财产保险
- **修缮费用界线**：以具体金额划分"小额维修"和"大修"；重大修缮的出资比例必须明确
- **使用禁止**：禁止宗教活动、禁止明火焚香、禁止未经审批修缮/改造/添建、禁止新增塑像
- **三级违约责任梯度**：一般违约（警告+整改）→严重违约（违约金+整改）→根本违约（立即解除+高额违约金）
- **监督检查**：定期检查（≥每月1次）+突击检查+书面记录+季度报告制度
- **交还程序**：15日内腾退+按交付时记录验收+逾期占用费+遗留物处置规则

### 11.3 低空经济/飞行营地合作协议
- **土地权属链条**：签约方声称的权利是否属实——农村土地归集体所有，个人不享有所有权
- **原租约限制**：原租约的用途（种植）和禁止条款（禁止改变用途、禁止建房）对合作开发的约束
- **用途变更审批**：农用地转建设用地须自然资源部门审批
- **原出租方同意**：转租/再流转须经原出租方书面同意并备案
- **集体决策程序**：涉及村集体土地的，须经村民代表大会决议+四议两公开
- **低空经济监管**：《民用航空法》（2026.7.1施行）首次确立低空经济法律地位
- **收益分配保障**：纯利润核算口径须明确；乙方须有查账权/审计权；建议设最低保底收益
- **合作期限**：不得超过原土地租约的剩余期限

### 11.4 技术服务合同/产品检测类
- **检测机构资质预审**：签约前须核查乙方CMA/CNAS认证资质的有效性和范围
- **检测报告质量保证条款（核心）**：乙方应对检测报告的专业性、科学性、准确性负责，明确赔偿机制
- **付款方式控制**：分阶段支付，至少保留20-30%尾款作为质量保证
- **违约责任具体化**：报告质量不合格、违反保密义务等违约情形的具体违约金标准
- **争议管辖选择**：优先约定甲方所在地或合同签订地法院管辖
- **异议/复检程序**：约定甲方对检测报告有异议时的申诉渠道和复检程序
- **后续配合义务**：检测报告后续使用中需乙方配合解释说明的，提前约定

### 11.5 融资租赁合同
- 租赁物选择权和瑕疵担保
- 租金构成（本金+利息+手续费等）
- 租赁期满后租赁物的归属
- 出租人对租赁物的所有权与买受人的善意取得

### 11.6 借款合同
- 利率约定不得超过法定上限（LPR四倍）
- 提前还款的处理
- 担保措施的完备性
- 企业间借贷的效力认定

### 11.7 担保合同
- 担保方式（一般保证/连带责任保证）的明确约定
- 保证期间和诉讼时效
- 公司对外担保的内部决议程序（《公司法》第15条）
- 抵押登记、质押交付的办理

### 11.8 股权转让合同
- 目标公司其他股东的优先购买权处理
- 股权权属保证（无质押、无冻结）
- 出资是否实缴的处理
- 交割后公司治理安排
- 国有资产、外资等特殊情形

### 11.9 股权代持合同
- 代持关系的合法性和合规性审查
- 实际出资人的权利保障机制
- 名义股东的违约风险防范
- 代持解除和显名化安排

### 11.10 合伙合同
- 合伙目的和经营范围
- 出资方式、数额和缴付期限
- 利润分配和亏损分担
- 入伙、退伙、除名条件和程序
- 合伙解散和清算
- 注意区分民事合伙与合伙企业

### 11.11 广告合同
- 广告内容合规审查义务的分配
- 知识产权归属（广告创意、设计成果）
- 发布期限和发布标准
- 广告效果评估标准

### 11.12 中介合同
- 中介服务的范围界定
- 中介成功的认定标准
- 委托人"跳单"的违约责任（民法典第965条）
- 中介费用的计算和支付条件

### 11.13 债权转让合同
- 债权是否可转让的审查（民法典第545条）
- 转让通知义务
- 从权利随主债权转移
- 债务人的抗辩权和抵销权

### 11.14 补充合同/变更协议
- 明确变更的范围和条款
- 与原合同的效力关系
- 除变更条款外，原合同其他条款继续有效
- 签署日期的正确处理

### 11.15 解除合同/终止协议
- 解除的法律依据（约定解除/法定解除）
- 解除后的财产返还和损失赔偿
- 尚未履行部分的处理
- 保密、争议解决等条款的存续
