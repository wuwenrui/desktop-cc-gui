## 二、劳动合同

### 2.1 签订时间要求
- 用工之日起1个月内签订书面合同（《劳动合同法》第10条）
- 超过1个月不满1年未签 → 每月支付**二倍工资**（第82条），最长11个月
- 满1年未签 → 视为已订立**无固定期限劳动合同**（第14条）

### 2.2 补签 vs 倒签
| 方式 | 落款日期 | 合同期限 | 法律效果 |
|------|----------|----------|----------|
| **补签** | 当前实际签署日 | 覆盖至入职之日 | 若劳动者未保留追索权利且系自愿，一般不产生双倍工资 |
| **补签（劳动者保留权利）** | 当前实际签署日 | 覆盖至入职之日 | 若劳动者补签时明确表示不放弃二倍工资，用人单位仍需支付（2025年川渝典型案例） |
| **倒签** | 人为填写为入职日 | =入职至今 | 违法，仍需支付双倍工资；空白合同"先签字后补填"视为未签（辽宁阜阳中院2025） |

**建议**：采用补签方式，合同期限从入职日起算，落款日期为实际签署日。补签时**应取得劳动者书面放弃二倍工资主张的声明**，否则仍存在被追索风险。杜绝空白合同操作。

### 2.3 劳动合同类型速查
| 人员类型 | 合同类型 | 社保 | 说明 |
|----------|----------|:----:|------|
| 正常劳动者 | 劳动合同 | 必须缴 | 标准用工 |
| 超龄人员（已享养老金） | 劳务合同（争议中） | 不缴养老/医疗，工伤保险强制参保 | 司法解释二（2025年9月施行）废止原"一刀切"劳务关系条款，转向"不完全劳动关系"，保障劳动报酬/休息休假/安全卫生/工伤四项核心权益；人社部《超龄劳动者基本权益保障暂行规定》征求意见稿已出 |
| 超龄人员（未享养老金） | 有争议 | — | 山西省高院（2017）晋民申2094号认定可为劳动关系 |
| 国企内退人员 | 劳务合同 | 不缴（原国企保留劳动关系） | 新单位客观上无法重复缴纳社保，经济补偿争议中诚信原则抗辩仍有适用空间 |
| 非全日制用工 | 劳动合同（可口头） | 工伤保险必须缴 | 每日≤4小时，每周≤24小时 |

### 2.4 定制合同关键条款
设计定制版劳动合同时应包含：
1. **工作地点条款**：预留跨区域调派灵活性（如"根据公司经营需要可在山西省内调整工作地点"）
2. **岗位调整条款**：明确合理的调岗范围
3. **薪酬拆分条款**：列明基本工资、岗位工资、绩效工资、补贴等科目
4. **工时条款**：结合实际工作模式设计
5. **保密条款**：涉及商业秘密、技术信息的保密义务
6. **违纪辞退引用条款**：明确引用公司《奖惩管理制度》作为处分依据

## 三、社会保险

### 3.1 基本原则
- 社会保险是**法定强制义务**，不因劳动者"自愿放弃"而免除（《劳动法》第72条、《社会保险法》第58条、第60条）
- 用人单位须为员工缴纳全部**五险**（养老、医疗、失业、工伤、生育保险），**选择性缴纳（如只缴三险不缴医保）同样属于"未依法缴纳社会保险费"**，员工可据此解除合同并主张经济补偿金
- 员工可随时投诉要求补缴，人社部门可责令补缴+罚款
- 员工可以"未依法缴社保"为由单方解约并主张经济补偿金（《劳动合同法》第38条、第46条）
- 城乡居民医保（原"新农合"）与职工医保**本质不同，不能互相替代**；城乡居民养老保险（原"新农保"）与职工养老保险亦属不同体系，退休后只能享受一份养老金（两体系合并计算、不重复享受）；员工有城乡居民社保（一卡两险：医疗+养老）不构成用人单位免缴职工社保的理由（深圳中院2025年判例：职工医保与居民医保报销差额由用人单位赔偿；赣州、瑞丽、泉州等多地人社部门明文规定用人单位不得以员工已参加城乡居民社保为由规避职工社保缴纳义务）

### 3.2 司法解释二第19条（2025年9月1日施行）—— 核心规则

《最高人民法院关于审理劳动争议案件适用法律问题的解释（二）》（法释〔2025〕12号）第19条确立三重机制：

| 机制 | 内容 |
|------|------|
| **① 无效认定** | 用人单位与劳动者约定或劳动者承诺无需缴纳社保的，**一律无效** |
| **② 经济补偿** | 未依法缴纳社保，劳动者可解除合同并主张**经济补偿金**；"诚信原则"不再能阻断该请求（最高法配套典型案例"朱某与某保安公司案"明确） |
| **③ 补贴返还** | 用人单位**依法补缴社保后**，可请求劳动者返还已支付的社会保险补偿（第19条第2款） |

> **重大变化**：2025年9月前各地裁判分裂——山东/江浙以"诚信原则"不支持经济补偿，北京/天津/吉林支持。司法解释二统一为：**不支持诚信原则抗辩，统一支持经济补偿**。

> **未解争议**：劳动者已在其他单位缴纳社保（如内退人员）导致客观上不能重复缴纳的，用人单位是否仍需支付经济补偿？目前司法实践中仍有分歧。

### 3.3 社保补缴争议的管辖
根据最高人民法院研究室法研[2011]31号答复（现行有效，2025年山东高院仍援引）：
- **民事法院不受理**社保补缴争议，社保征缴属于行政职责
- 员工须通过人社部门行政投诉渠道主张补缴
- 例外：用人单位未办理社保手续且社保经办机构不能补办，导致劳动者无法享受社保待遇的，法院**应当受理**（赔偿损失）

### 3.4 社保补贴确认书的设计要点与法律边界

**设计要点**：
- 明确员工本人拒绝缴纳的事实
- 明确公司已将单位应承担费用以补贴形式发放（须为工资之外的**真正额外支出**，而非工资换标签）
- 约定员工事后主张补缴则须退还全部已领补贴（司法解释二第19条第2款赋予法定追回权）

**法律边界**：
1. **不能免除补缴义务**：即使签署确认书，用人单位仍需依法补缴社保
2. **不能阻挡经济补偿**：司法解释二第19条已封堵"诚信原则"抗辩路径
3. **追回补贴的前提**：用人单位须**先完成补缴**，才能请求员工返还补贴
4. **追回范围有限**：若"补贴"只是将原有工资换标签（总金额未增加），法院不认定为真正补贴，追回请求可能不被支持（湖南永州2024判例）
5. **核心价值重新定位**：①补缴后追索已发补贴款的法定依据（第19条第2款）；②行政调查时证明非恶意不缴

### 3.5 工资拆分社保金额是否可行？

**结论：不可行。** 将社保费明示计入工资（含单位和个人部分）替代实际缴纳，约定一律无效。

| 判例 | 核心结论 |
|------|----------|
| 湖南永州 (2024)湘1103民初6912号 | 6800元拆为"3000基本+900社保补贴"，总金额不变 → 拆分不成立，按全额算补偿金 |
| 深圳仲裁 2025 | 8000元拆三笔含"社保补贴1500"→ 认定就是工资组成部分 |
| 北京一中院 2026.05 | 按低于实际工资基数缴社保、差额发补贴 → 约定无效 |
| 西安莲湖法院 2024 | 员工签字领社保补助5万余元 → 约定无效，但员工须返还补贴 |

**区分关键**：
- 工资换标签（总金额不变）→ 法院不认可为补贴 → 连追回都做不到
- 工资外真正额外支出 + 员工签字 → 约定仍无效，但补缴后可依第19条第2款追回

### 3.6 雇主责任险
- 劳务关系人员（超龄、内退）不能参加工伤保险的，建议单独购买**雇主责任险**
- 超龄人员新规（司法解释二 + 人社部征求意见稿）要求**强制参保工伤保险**，待正式文件出台后以新规为准

### 3.7 城乡居民社保（原"新农合"/"新农保"）—— 与职工社保的关系

**制度沿革**：
- 原"新农合"（新型农村合作医疗）于2016年整合为**城乡居民基本医疗保险**（国发〔2016〕3号）
- 原"新农保"（新型农村社会养老保险）于2014年整合为**城乡居民基本养老保险**（国发〔2014〕8号）
- 两险合并体现在**一张社保卡**上（一卡两险：医疗+养老），居民60岁起可按月领取养老金

**与职工社保的本质区别**：

| 维度 | 城乡居民社保 | 职工社保 |
|------|------------|---------|
| 参保性质 | 自愿参保 | 强制参保 |
| 缴费方式 | 个人按年缴费+政府补贴 | 单位+个人按月共同缴费 |
| 医保报销比例 | 较低（通常50-70%） | 较高（通常70-90%） |
| 养老保险 | 个人账户+基础养老金（较低） | 统筹账户+个人账户（较高） |
| 工伤保险 | 无 | 有（用人单位缴费） |
| 失业保险 | 无 | 有 |
| 生育保险 | 部分地区并入医保 | 有（已并入职工医保） |

**关键规则**：
1. **不能互相替代**：员工有城乡居民社保（无论医疗还是养老），均不构成用人单位免缴职工社保的合法理由。赣州、瑞丽、泉州等多地人社部门明文强调此规则。
2. **退休时合并计算**：根据人社部发〔2014〕17号，职工养老保险缴费满15年的，城乡居民养老保险转入职工养老保险；职工养老保险不满15年的，职工养老保险转入城乡居民养老保险。**同一时段不能重复享受两份养老金**，重复缴费时段只计算职工养老保险年限，城乡居民养老保险个人账户退还。
3. **工伤风险**：城乡居民社保不含工伤保险。员工发生工伤时，若用人单位未参加职工工伤保险，将**全额自行承担**工伤保险待遇。
4. **医保报销差额赔偿**：深圳中院2025年判例——用人单位未缴职工医保，员工以居民医保报销后，差额部分由用人单位赔偿。

> **实务提示**：面对员工"我已经有农合了，不用再缴社保"的主张，应向员工解释：①两者参保性质不同（自愿vs强制），用人单位无法定选择权；②职工医保报销比例更高、保障更全；③没有工伤保险是最大隐患，一旦发生工伤用人单位和员工均无保障；④退休后职工养老金远高于居民养老金。

### 3.8 未缴医保的医疗费赔偿计算

**核心原则：差额赔偿，非全额赔偿。**

用人单位未缴职工医保，导致员工患病无法享受职工医保待遇的，赔偿的是"本可报销金额的损失"，而非全部医疗费。

**赔偿计算公式**：
```
赔偿总额 = 按职工医保标准可报销的统筹支付金额 - 员工已通过城乡居民医保等其他途径获得的报销金额
```

**不属于赔偿范围**（员工自行承担）：
- 起付线以下部分
- 封顶线以上部分
- 个人自付比例部分
- 全自费项目/药品（如人血白蛋白等）

**关键判例**：

| 判例 | 核心要点 |
|------|----------|
| (2025)渝0153民初13385号（重庆荣昌，2026.2） | 员工签有《自愿放弃购买社保承诺书》+月领1200元补贴，患病后居民医保报销37,094.97元，职工医保本可报销45,648.76元 → 法院仅判赔**差额8,553.79元**，非全部医疗费。《承诺书》被认定无效 |
| 常州中院2024年度典型案例 | 法院函询医保基金经办机构后，仅支持医保统筹基金支付部分11,991.55元 |

**实操步骤**（员工发生医疗费索赔时）：
1. 要求员工提供：医疗费发票原件、费用明细清单、医保结算单（如有）、出院小结、病历
2. **立即向当地医保局发函**查询："假如该员工参加职工医保，本次住院可报销金额为多少"
3. 以医保局官方核算作为赔偿上限，而非按员工发票总额赔付
4. 员工已通过居民医保报销的部分应从赔偿额中扣除（居民医保客观上起到减损作用）
5. 基于核算结果与员工协商签署书面和解协议

> **注意区分**：工伤与医疗不同。未缴工伤保险发生工伤的，用人单位需全额承担工伤保险待遇（商业工伤保险仅具补偿性质，超出部分仍由单位承担）。未缴医保发生疾病的，按上述公式差额赔偿。

### 4.1 标准工时
- 每日不超过8小时，每周不超过40小时（《劳动法》第36条）
- 每周至少休息1日（第38条）

### 4.2 加班费计算
| 加班类型 | 支付标准 | 可否补休替代 |
|----------|:--------:|:------------:|
| 工作日延时 | 150% | **不可** |
| 休息日加班 | 200% | **可优先补休**（决定权在用人单位，非员工选择；补休安排须及时，不得无限期拖延） |
| 法定节假日 | 300% | **不可** |

**加班费计算基数约定规则**：可在劳动合同/制度中明确约定以**基本工资**为计算基数，但须同时满足四个条件：

1. **约定明确具体**（在劳动合同或规章制度中载明）
2. **不低于当地最低工资标准**（违反则约定无效）
3. **实质公平合理**（基数不能畸低——法院采用"实质重于形式"审查，恶意拆分不被认可）
4. **劳动者签字确认**

| 判例 | 核心结论 |
|------|----------|
| 青岛某公司案（2026，仲裁→高院全驳回） | 劳动合同约定"加班费按基本工资计算"，员工主张按实际工资补发5年差额，四级裁判均驳回——约定有效 |
| 上海2024年判例 | 8000元拆为3000基本+津贴+绩效，法院认定恶意拆分，以8000元全额为基数 |
| 天津保税区仲裁（2025） | 基本工资1400元低于最低工资→按最低工资标准作为基数 |

**补休规则要点**：
- 休息日加班：用人单位享有**优先安排补休的决定权**（《劳动法》第44条第(二)项"又不能安排补休的"），非员工选择权。安排补休应及时，不得无限期拖延
- 工作日延时、法定节假日加班：**必须支付加班费**，不得以补休/调休替代

### 4.3 综合计算工时工作制
- 适用于因生产特点无法实行标准工时的企业
- 需向劳动行政部门**申请批准**
- 山西省晋劳社厅发[2008]25号：制造业"受市场因素影响，生产任务不均衡企业的职工"纳入适用对象
- 周期内总工时不超过法定标准即可，有效降低日常加班费支出

### 4.4 不定时工作制
- 适用于高管、外勤、销售、值班等无法按标准工时衡量的岗位
- 也需向劳动行政部门审批

## 六、劳动合同解除与经济补偿

### 6.1 经济补偿金速查
| 解除类型 | 法律依据 | 补偿标准 |
|----------|----------|:--------:|
| 协商一致（用人单位提出） | 第36条 | N |
| 员工因单位违法单方解除 | 第38条 | N |
| 无过错性辞退（不能胜任等） | 第40条 | N+1 |
| 经济性裁员 | 第41条 | N |
| 合同期满终止（单位不续签） | 第44条、第46条 | N |
| 违法解除/终止 | 第48条、第87条 | **2N** |

> N = 每满1年支付1个月工资，6个月以上不满1年按1年计，不满6个月计0.5个月
> 上限：月工资>当地社平工资3倍的，按3倍计算，最高12年（第47条）

### 6.2 过错性辞退（第39条）—— 不支付补偿金
需满足以下情形之一：
1. 试用期不符合录用条件（需有明确的录用条件在先）
2. 严重违反用人单位规章制度（需有民主程序制定+公示的制度+明确"严重"情形）
3. 严重失职、营私舞弊，给单位造成重大损害
4. 劳动者同时与其他单位建立劳动关系，对完成本单位工作造成严重影响
5. 以欺诈手段使单位在违背真实意思下订立合同
6. 被依法追究刑事责任

**过错性辞退的处分程序（缺一不可）**：
1. **调查取证**：客观全面收集书面记录、监控、证人证言等证据
2. **书面告知**：将违纪事实、拟处分内容书面通知员工
3. **给予申辩机会**：保障劳动者的程序参与权与抗辩权
4. **事先通知工会**：《劳动合同法》第43条——用人单位单方解除劳动合同，应当**事先将理由通知工会**。公司无工会的，应通过听取职工代表意见等方式履行替代性告知程序。**遗漏此步构成程序违法，即便实体理由充分也会导致解除无效**
5. **作出决定并送达**：解除通知须有效送达劳动者，解除以送达为生效要件

> **制度建设 + 程序合规 + 证据留存**三要素缺一不可。规章制度未经民主程序制定或未公示→处分依据无效；未通知工会→程序违法→2N赔偿；证据不足→事实认定不成立→2N赔偿。

### 6.3 无过错性辞退（第40条）—— 支付N+1
1. 医疗期满后不能从事原工作，也不能从事另行安排的工作
2. **不能胜任工作**，经培训或调岗后仍不能胜任 → 需走完"培训/调岗→二次考核不合格"的完整程序
3. 客观情况发生重大变化，致使合同无法履行，协商变更不成

### 6.4 工作地点变更（搬迁）的处理
判定是否构成"客观情况重大变化"（第40条第3项）的要素：
- **跨市搬迁**+无合理补救措施（交通、住宿补贴等）→ 很可能构成重大变化 → 协商不成可单方解除，支付N+1
- **同城小幅搬迁**+提供通勤补贴 → 不构成重大变化 → 员工拒绝随迁可按违纪处理
- 判定时综合考虑：距离、是否有补救措施、搬迁的必要性

### 6.5 离职协议的关键条款
协商一致解除劳动合同时，离职协议应包含：
1. **明确提出方**（谁提出的解除，决定是谁付N还是N+1还是无需支付）
2. **分项列明补偿金额**（工资结清、经济补偿金、未休年假折薪等）
3. **"无争议"条款**（双方确认再无任何劳动争议）
4. **"退还/追索"条款**（如员工已领社保补贴的处理）
5. **税务处理**：经济补偿金在规定限额内（≤当地上年社平工资3倍×工作年限）免征个税（财税[2001]157号）

## 七、规章制度制定（民主程序）

### 7.1 法律要求
《劳动合同法》第4条：直接涉及劳动者切身利益的规章制度须经：
1. 职工代表大会或全体职工**讨论**，提出方案和意见
2. 与工会或职工代表**平等协商**确定
3. **公示**或告知劳动者

**未经上述程序的制度，不能作为处分员工的依据**（仲裁/诉讼中不被采信）。

### 7.2 职工代表大会要求（总工发[2012]12号《企业民主管理规定》）
- 代表人数：≥全体员工**5%**，且绝对下限**不少于30人**（第8条）
- 管理层比例：中层以上管理人员和领导人员**≤20%**（第9条）
- **职工不足50人的企业，应召开全体职工大会**，无需选举职工代表
- 表决门槛：≥2/3代表出席，>1/2全体代表通过
- 重要事项：**无记名投票**表决

### 7.3 公示告知的合规方式
- 人手一册签收（最稳妥）
- 公告栏公示 + 拍照留存
- 邮件/微信群发送 + 截图留存
- 培训考试 + 签到表 + 试卷存档
- **关键词**：员工签字确认 + 过程记录留存

## 八、工资管理

### 8.1 工资拆分结构
推荐拆分科目：
- 基本工资（加班费计算基数、经济补偿金计算基数，须有实质依据，不低于最低工资）
- 岗位工资
- 绩效奖金（须有真实考核制度且经员工确认，不能仅作为降低加班费基数的"口袋项目"）
- 加班工资
- 全勤奖
- 其他补贴（工龄、餐补、交通等）

> **司法边界**：法院穿透工资结构的名义，审查各项是否实质属于正常工作时间工资。固定津贴、无实质考核的常规绩效均可能被认定为正常出勤工资组成部分，计入加班费基数。调整工资结构（降固定、增浮动）须经协商一致，单方调整即便总额不变也可能被认定为变相降薪（北京三中院 (2024)京03民终3254号）。

### 8.2 计算基数策略
- **加班费基数**：劳动合同或制度中明确约定以"基本工资"为计算基数，但基本工资须合理、有实质依据，不得为规避加班费而人为压低
- **经济补偿金基数**：离职前12个月平均工资（含奖金、补贴等全部货币收入）
- 拆分后基本工资降低 → 加班费支出降低，但须确保拆分有实质合理性和员工确认

### 8.3 违纪处分与经济扣除的合规边界

**"罚款"概念已于2008年废止**：《企业职工奖惩条例》于2008年1月15日被国务院令第516号废止，用人单位自此丧失对员工"罚款"的法律依据。用人单位不是行政机关，无权实施行政处罚性质的罚款。

**20%扣除规则仅适用于"赔偿经济损失"**（《工资支付暂行规定》第16条）：
- 适用前提：因劳动者本人原因给用人单位造成**实际经济损失**
- 须在劳动合同中有明确约定
- 每月扣除≤当月工资的**20%**
- 扣除后剩余工资≥当地**最低工资标准**
- **不适用于一般违纪**（如迟到、早退）——不能借此规定对违纪行为直接扣工资

**合规替代方案（推荐）**：
- 将违纪与**绩效工资/奖金/全勤奖**挂钩（浮动考核制）——"当月有严重违纪行为的，取消当月绩效奖金/全勤奖"
- 此属于绩效管理范畴，法律依据充分，仲裁认可度高
- 须在规章制度中明确违纪行为与绩效/奖金的对应扣减规则，且制度经民主程序+公示

> 江苏、深圳等地有地方性规定允许有限度的违纪经济处分（江苏≤20%、深圳≤30%），但**山西省无此类规定**，在山西直接用"扣工资"方式处罚违纪，合法性基础薄弱。

## 九、常见风险速查

### 9.1 双倍工资风险
| 情形 | 风险 |
|------|------|
| 入职超1月未签合同 | 最长11个月双倍工资差额 |
| 合同期满未续签 | 同上（到期次日起算） |
| 补签合同但落款日期倒填 | 仍有双倍工资风险 |

### 9.2 违法解除风险
- 没有制度依据的辞退 → **2N**赔偿
- 没有走完法定程序的"不能胜任"解除 → **2N**
- 没有留存证据的过错性辞退 → **2N**
- **处分程序遗漏"通知工会"**（劳动合同法第43条）→ 程序违法 → **2N**
- 以"罚款"替代绩效管理 → 扣款依据不足 → 可能被认定克扣工资

### 9.3 社保用工风险
- 自愿放弃社保 → 仍不免除单位缴纳义务（司法解释二第19条，2025年9月起全国统一）
- 不缴社保 → 员工随时可单方解除+主张经济补偿金（"诚信原则"抗辩已失效）
- **只缴三险不缴医保 → 同样属于"未依法缴纳"**，员工可解除合同+主张经济补偿；员工就医损失按职工医保与居民医保的报销差额赔偿（非全额医疗费），见3.8节赔偿公式
- 以商业保险替代社保 → 不合法
- 以现金补贴替代社保 → 约定无效，补缴社保后可要求返还补贴（第19条第2款）
- 工资拆分社保金额替代缴纳 → 不可行，法院不认可
- 城乡居民社保（原"新农合"）不能替代职工社保 → 一卡两险（医疗+养老）均属不同体系，不能互相替代；发生工伤用人单位全额承担

### 9.4 关联公司混同用工
- 多主体交叉用工须明确各公司用工主体地位
- 避免同时为多家公司提供劳动但只有一家签合同
- 避免工资发放主体与社保缴纳主体不一致

## 十、执法标准口径（2026年5月版本）

本 Skill 在执行法律分析、风险评估与文件审查时，遵循以下统一标准口径：

1. **工作日、月薪制**：月计薪天数按 21.75 天/月 计算，日工资 = 月工资 ÷ 21.75。
2. **员工入职登记**：可要求填写入职登记表，并注明入职日期、试用期、部门/岗位、教育背景、工作经历、紧急联系人等。同时应设置"员工确认"签字栏，载明"本人保证以上填写信息真实、完整，如有虚假或隐瞒，自愿承担一切法律责任，接受公司按照严重违纪解除劳动合同并不支付任何补偿。"
3. **试用期上限**：合同期限3个月以上不满1年，试用期≤1个月；1年以上不满3年，试用期≤2个月；3年以上固定期限或无固定期限，试用期≤6个月。
4. **试用期工资下限**：≥（转正后约定工资 × 80%），且 ≥ 当地最低工资标准。
5. **竞业限制期限与补偿下限**：竞业限制≤2年；补偿标准 ≥ 劳动合同终止前12个月平均工资 × 30%，且 ≥ 当地最低工资标准。
6. **加班费计算**：法定节假日加班必须支付 300% 工资，不得以补休替代。休息日加班用人单位可优先安排补休（决定权在用人单位），无法补休的支付200%。工作日延时加班支付150%，不可补休替代。加班费基数可在劳动合同或制度中约定以基本工资为准，须满足四条件：明确约定+不低于最低工资+实质公平合理（不能畸低）+劳动者签字确认。青岛2026年判例确认约定有效（仲裁→高院全驳回员工请求）。
7. **经济补偿基数封顶**：月工资超过本地区上年度职工月平均工资3倍的，按3倍计算，支付年限最高不超过12年。
8. **规章制度民主程序两步走**：①职工代表大会或全体职工讨论（留签到表和会议记录）；②工会或职工代表协商确定（留协商记录）。两步缺一不可。职工代表人数≥全体职工5%且绝对不少于30人（总工发[2012]12号第8条）；中层以上管理人员≤代表总数20%（第9条）。**职工不足50人的企业，应召开全体职工大会**，无需选举职工代表。
9. **公示告知可采复合方式**：人手一册签收表、公告栏张贴拍照、微信/钉钉工作群发送截图、培训签到考核等方式。
10. **部分人员劳务合同处理**：已领取养老金的超龄返聘人员，按司法解释二（2025年9月施行）已废止原"一刀切"劳务关系规则，转向"不完全劳动关系"——保障劳动报酬/休息休假/安全卫生/工伤四项核心权益，但不享有经济补偿金、无固定期限合同、二倍工资；工伤保险为强制参保项目。从国企内退到私企工作的人员签署劳务合同，因原国企保留劳动关系，新单位客观上无法重复缴纳社保，经济补偿争议中诚信原则抗辩仍有适用空间。
11. **年休假仲裁时效**：从应休假年度次年的 1 月 1 日起计算 1 年。
12. **社保补贴确认书**：约定"如员工事后主张补缴社保，则须将已领取的社保补贴全额退还公司"。司法解释二第19条第2款已赋予法定追回权，但追回前提是单位先完成补缴，且补贴须为工资之外的真正额外支出（非工资换标签）。
13. **城乡居民社保（原"新农合"）不能替代职工社保**：原"新农合""新农保"已分别整合为城乡居民基本医疗保险和城乡居民基本养老保险（一张社保卡、一卡两险）。无论医疗还是养老，均与职工社保属不同体系，不能互相替代。员工有城乡居民社保不构成用人单位免缴职工社保的合法理由。退休时两体系合并计算，同一时段不重复享受两份养老金。多地人社部门（赣州、瑞丽、泉州）明文规定用人单位不得以员工已参加城乡居民社保为由规避职工社保缴纳义务。
14. **关联企业混同用工**：司法解释二第3条——关联单位交替用工可被要求共同承担支付劳动报酬、福利待遇等责任。可通过在劳动合同、工资发放、社保缴纳、日常管理等环节统一以一家公司名义来处理，规避该合规风险。
15. **未缴职工医保的医疗费赔偿计算**：赔偿范围非"全额医疗费"，而是差额赔偿。公式：赔偿总额 = 按职工医保标准可报销的统筹支付金额 - 员工已通过居民医保等其他途径获得的报销金额。起付线以下、封顶线以上、个人自付比例部分及全自费项目由员工自担。(2025)渝0153民初13385号（2026.2）：员工有居民医保+签放弃承诺书+领社保补贴，法院仅判赔差额8,553.79元。实操：员工索赔时立即向医保局发函查询"如参加职工医保可报销金额"，以官方核算为赔偿上限。
16. **过错性辞退处分程序五步走**：调查取证→书面告知→给予申辩机会→事先通知工会（《劳动合同法》第43条，无工会的应通过职工代表替代程序）→作出决定并送达。遗漏通知工会构成程序违法，即便实体理由充分也导致解除无效→2N赔偿。
17. **用人单位无罚款权**：《企业职工奖惩条例》已于2008年废止，企业不是行政机关，无权对员工实施罚款。违纪管理应通过绩效奖金/全勤奖挂钩机制实现（浮动考核），而非直接扣工资。20%工资扣除上限（《工资支付暂行规定》第16条）仅适用于赔偿实际经济损失，不适用于一般违纪。山西省无地方法规支持违纪扣款。

## 十二、劳动合同中止制度

### 12.1 概念与定义

劳动合同中止，指在劳动合同履行过程中，因法定或约定情形导致劳动合同暂停履行，但**劳动关系保留**的一种特殊状态。中止期间劳动者停止提供劳动，用人单位停止支付劳动报酬，中止期间不计入本单位工作年限。

> **与劳动合同解除的本质区别**：中止是"暂停"（劳动关系保留），解除是"终止"（劳动关系消灭）。本案中赵美玉系主动辞职导致解除，非中止。

### 12.2 法定中止情形（综合全国各地立法）

| 中止情形 | 法律依据 | 说明 |
|----------|---------|------|
| 协商一致中止 | 山东省条例第26条第1项、江苏省条例第30条第1项、安徽省条例第22条等 | 须以书面形式明确约定。案例：东台市法院(2019)苏0981民初5406号认定"保留劳动关系协议"属合法中止 |
| 涉嫌违法犯罪被限制人身自由 | 山东省条例第26条第2项、江苏省条例第30条第2项、劳部发〔1995〕309号第28条 | 适用条件：被收容审查、拘留、逮捕、留置等**限制人身自由**措施。纪委调查但未被留置的不必然构成此情形 |
| 不可抗力 | 山东省条例第26条第3项、各地条例 | 疫情不得作为中止理由（人社部发〔2020〕17号明确禁止）。停产解散等经营困难情形，法院有认定中止的案例 |
| 应征入伍或履行法定义务 | 江西省解答第10条第4项等 | |
| "长期两不找" | 北京、吉林、江苏等地司法实践 | 须满足长期不提供劳动、不支付报酬、不管理人事档案的"三不"状态 |
| 其他法定情形 | 各地条例兜底条款 | |

### 12.3 中止的法律效力

| 项目 | 处理规则 | 依据/案例 |
|------|---------|-----------|
| 劳动报酬 | 用人单位可停止支付 | 山东条例第26条、江苏条例第30条 |
| 社会保险 | **可停止缴纳**（各地明确程度不同，详见12.4） | 山东、江苏、江西、上海等地均有明确规定 |
| 工作年限 | 中止期间不计入本单位工龄 | 江苏盐城中院(2021)苏09民终2855号 |
| 竞业限制 | 仍可生效 | 东台法院(2019)苏0981民初5406号 |
| 恢复履行 | 中止情形消失后应恢复，无法履行的除外 | 各地条例 |

#### 地方规定对比表

```
山东/江苏/江西：劳动合同中止期间→劳动关系保留→劳动合同暂停履行
→用人单位可以不支付劳动报酬并停止缴纳社会保险费

上海：劳动合同中止履行→办理社会保险帐户暂停结算（封存）手续

北京：劳动关系中断→社会保险关系随之中断→缴费工资基数为零
→不能以最低工资或社会平均工资为基数补缴基本养老保险费

山西：协商一致→中止履行→双方暂停履行劳动合同的有关权利和义务
→未明确社保处理
```

### 12.5 学术观点与理论争议

核心论文：《论劳动合同中止期间用人单位的社保缴纳义务》（2026年4月）

| 问题 | 无争议结论 | 争议焦点 |
|------|-----------|---------|
| 个人缴费部分 | 属于劳动报酬的一部分，中止期间用人单位无需支付劳动报酬，故**无需代缴个人部分** | — |
| 单位缴费部分 | — | 理论上学界分两派：一派认为劳动关系存续即应继续缴纳（《社会保险法》第60条"非因不可抗力不得减免"）；另一派认为权利义务对等，劳动者不提供劳动则用人单位不承担义务。实践困境：劳社险中心函〔2006〕60号规定中止期间缴费工资基数为零，单位与个人应缴金额均为零，单独缴纳单位部分"无现实基础" |

**论文结论性建议**：参照上海市做法，建立社保账户**封存**制度，中止期间暂停结算、不计缴费年限；中止情形消失后允许补缴并明确补缴规则。

### 12.7 "举重以明轻"原则：中止 vs 解除的社保义务比较

```
劳动关系状态层级（从强到弱）：
正常履行 → 劳动合同中止（劳动关系保留） → 劳动合同解除（劳动关系消灭）

各地法律对"中止"的社保规定：多数地区允许停缴

推断：
在"劳动关系保留"的中止状态下法律都允许停缴社保
那么在"劳动关系消灭"的解除状态下，用人单位更无缴纳义务
```

## 十三、劳动者辞职制度

### 13.1 单方预告解除权（形成权）

**《劳动合同法》第37条**：劳动者提前30日以书面形式通知用人单位，可以解除劳动合同。劳动者在试用期内提前3日通知用人单位，可以解除劳动合同。

**核心法理——形成权**：
- 解除的意思表示一经送达用人单位即发生法律效力
- 无须用人单位同意或批准
- 30日预告期届满后劳动合同当然解除
- 用人单位"不予受理"或"不批准"的表态不影响解除效力

**关键裁判观点**：
- 北京市第二中级人民法院(2023)京02民终9373号：劳动者辞职属形成权，离任审计未完成不影响出具离职证明的义务
- 山西省吕梁市中级人民法院(2025)晋11民终1000号：劳动关系解除系形成权，意思表示到达对方时即形成解除

### 13.2 预告期的计算

| 发出方式 | 送达日认定 | 30日届满日 |
|----------|-----------|-----------|
| 当面提交 | 提交当日 | 提交日+30日 |
| 微信/邮件 | 发送并到达对方系统之日 | 发送日+30日 |
| EMS邮寄 | 签收日（或拒收日） | 签收/拒收日+30日 |

### 13.3 辞职与实际提供劳动的关系

辞职后30日预告期内劳动者仍在职，应正常提供劳动。如劳动者在预告期内请病假，用人单位应按病假工资规定支付病假工资至预告期满之日。预告期满后劳动合同解除，用人单位无继续支付薪酬的义务（包括病假工资）。

### 13.4 辞职与离职手续的剥离

- **劳动合同解除的法律效力**：不可阻挡，30日预告期满即生效
- **离职行政手续的办理**（出具解除证明、档案转移、社保转出）：可在法定时限内办理，但基于特殊情形（如离任审计、纪检监察调查）可暂缓
- 司法实践认可暂缓办理行政手续的合法性：河南省郑州市中级人民法院(2024)豫01民终8430号（孔某案）

## 十四、国有企业管理人员的特殊规定

### 14.1 《国有企业管理人员处分条例》（国务院令第781号，2024年9月1日施行）

#### 适用范围（第2条）

"国有企业管理人员"的三类界定：

| 项 | 描述 | 适用分析 |
|---|------|---------|
| 第(一)项 | 在**国有独资、全资公司、企业**中履行管理职责的人员 | ❌ 混合所有制企业不适用 |
| 第(二)项 | 经**国有独资、全资公司、企业**提名/任命，在**国有控股、参股公司**任职的人员 | ⚠️ 适用以提名方为国有独资/全资为前提 |
| 第(三)项 | 经**国家出资企业**中管理国有资产的**组织**批准/决定，在国有控股/参股公司任职的人员 | ⚠️ "国家出资企业"范围更宽，含国有控股上市公司 |

> **实务提示**：如果股东方是上市公司的全资子公司（而非直接国有独资），第(二)项适用存在争议。应从第(三)项寻找适用空间——审查赵美玉的任命是否经过了中国中铁体系中管理国有资产的组织批准/决定。

#### 被调查期间的特殊限制（第34条）

- 适用前提：①当事人被认定为"国有企业管理人员"；②**已被正式立案调查**
- 效果：未经决定立案的任免机关、单位同意，不得出境、辞去公职
- 限制办理：不得交流、晋升、奖励或办理退休手续
- 不直接限制：劳动合同解除的法律效力（劳动法层面的解除权不受处分条例限制）

#### 初步核实与立案调查的区别

| 阶段 | 法律效果 |
|------|---------|
| 问题线索初步核实（初核） | 尚不构成"已被立案调查"，用人单位不得援引第34条限制辞职 |
| 正式立案调查 | 正式立案后，当事人属于"已被立案调查"状态，可援引第34条 |
| 调查终结 | 根据结论作出处分或撤销案件 |

### 14.2 纪检/监察调查期间的薪酬处理

**人社厅发〔2020〕54号函**（人力资源社会保障部办公厅关于国有企业负责人涉嫌违纪违法被调查期间薪酬支付问题有关意见的函）：

| 情形 | 薪酬处理 |
|------|---------|
| 被留置/拘留/逮捕（限制人身自由） | 停止支付全部薪酬 |
| 被调查但未被采取强制措施 | 暂缓支付绩效、奖金等浮动薪酬；按不超过本人当年基本年薪月发放标准计发生活费 |
| 调查认定无违纪违法 | 补发应发薪酬 |
| 调查认定存在违纪违法 | 依据处理结果扣减薪酬 |

**适用对象**：国有企业负责人（含法定代表人、董事长、副董事长、董事、总经理、副总经理、党委书记、副书记、纪委书记、工会主席等）

### 14.3 调查期间的劳动关系与社保处理

**关于社保——全国层面尚无明确规定**，各地实践存在差异：

| 情形 | 主流实务处理 |
|------|-------------|
| 被**留置**（限制人身自由） | 江苏、上海、江西等地明确可停缴社保（劳动合同中止）；无明确规定的地区建议继续缴纳或封存 |
| 仅被**立案调查**（未被留置） | 社保争议较大，多数建议继续缴纳以降低风险。薪酬可暂缓支付并发放生活费 |
| 调查后**无问题** | 补发薪酬，社保按应缴状态处理 |
| 调查后**有问题且开除** | 解除劳动关系，社保关系转移或封存 |

**个人部分 vs 单位部分区分处理原则**（学术界基本共识）：
- 个人部分：属于劳动报酬组成部分，无工资则无代扣基础，原则上不应缴纳
- 单位部分：以劳动关系存续为前提，各地规定不同，实践中争议较大

## 十五、重要原则与操作口诀

### 15.1 关键法律原则速记

| 原则 | 内容 | 应用场景 |
|------|------|---------|
| 辞职形成权 | 劳动者辞职无须用人单位同意，30日届满即解除 | 员工辞职、高管离职 |

| 举重以明轻 | 劳动合同中止（关系保留）可停缴社保→解除（关系消灭）更无义务 | 社保处理方案设计 |

| 社保法定不可约定放弃 | 《社会保险法》第60条、司法解释二第19条 | 社保争议 |
| 通知工会不可遗漏 | 《劳动合同法》第43条 | 单方解除程序 |

## 十六、员工涉刑情况下劳动关系处理

### 16.1 "被依法追究刑事责任"的认定

《劳动合同法》第39条第6项解除的前提：

| 情形 | 是否构成"被追究刑事责任" | 对劳动关系的影响 |
|------|------------------------|-----------------|
| 被人民法院判处刑罚（含主刑和附加刑，含**缓刑**） | **是** ✅ | 可依第39条第6项解除 |
| 被人民法院免予刑事处罚 | **是** ✅ | 可依第39条第6项解除 |
| **酌定不起诉**（刑诉法第177条第2款） | **否** ❌ | 不可依第39条第6项解除；可依第39条第2项（严重违纪）解除 |
| 法定不起诉（刑诉法第16条，实质无罪） | **否** ❌ | 应恢复劳动关系，补发中止期工资 |
| 证据不足不起诉（刑诉法第175条第4款） | **否** ❌ | 应恢复劳动关系，补发中止期工资 |

> **核心依据**：劳社厅函〔2003〕367号明确酌定不起诉不属于"被依法追究刑事责任"，用人单位不能据此解除。
> **缓刑明确可解除**：被判处缓刑仍属于"被依法追究刑事责任"，用人单位可解除劳动合同。法律赋予选择权（也可留用但须调岗签协议）。

### 16.2 限制人身自由期间的劳动关系处理

**劳部发〔1995〕309号第28条**：劳动者被收容审查、拘留或逮捕的，用人单位可暂时停止劳动合同履行，不承担相应义务。

| 处理事项 | 规则 |
|----------|------|
| 工资 | 可停止支付（全国性规则）。各地区规定：北京/天津/深/吉可不支付；上海/江苏/山东/江西可停发 |
| 社保 | 上海可封存、江苏/江西可停缴、**北京实践须继续缴纳** |
| 考勤 | 因被限制人身自由无法出勤，不宜按旷工处理（(2014)萍民一终字第165号） |
| 奖金 | 未提供劳动期间无权获得奖金（(2018)豫96民再30号） |

### 16.3 取保候审期间的劳动关系处理

取保候审**不属于法定的单方解除情形**，不可以此为由直接解除。

| 情形 | 工资处理 |
|------|---------|
| 正常上班（提供劳动） | 应按约定支付工资（北京、贵阳、辽宁、山东、江西、浙江等均支持） |
| 上班但调岗 | 按新岗位标准支付，不低于最低工资 |
| 不上班但未解除 | 重庆需发生活费；浙江暂停支付 |
| 协商待岗 | 按协议约定支付 |

### 16.4 开除/解除操作程序

```
获取生效法律文书 → 通知工会（法定程序，不可省略！） 
→ 合理期限内发出解除决定 → 有效送达 
→ 15日内办理离职手续
```

**关键判例**：(2020)京03民终10244号——事隔近2年才解除被认定为违法解除。建议收到生效判决后**7日内**启动工会程序。

### 16.5 不起诉后的恢复义务

- 法定不起诉/证据不足不起诉：**应恢复劳动关系**，补发中止期间工资（员工通过《国家赔偿法》向有关部门追偿损失）
- 酌定不起诉：不可依第39条第6项解除，建议通过规章制度解除或协商解除

## 十八、司法解释二（法释〔2025〕12号）核心规则

**2025年2月17日通过，2025年7月31日公布，2025年9月1日施行，共21条。**

### 18.1 混同用工——关联单位共同担责（第3条）

多个关联单位可**共同承担**支付报酬、福利等责任。针对"一套人马、多块牌子"的混同用工。

### 18.2 二倍工资新规（第6-9条）

| 项目 | 规定 |
|------|------|
| 计算方式 | 按月计算，不满一月按**实际工作日**计算 |
| 最长时限 | 11个月 |
| 用人单λ免责 | 不可抗力；劳动者故意或重大过失未签；其他法定情形 |
| 自动续延 | 孕期/医疗期/服务期等，合同自动续延不视为未签 |

### 18.3 无固定期限合同——防止规避（第10条）

以下情形视为"连续订立二次"：
- 协商延长合同期限累计达**一年以上**
- 约定合同期满后**自动续延**
- 变换签约主体但仍在原岗位/场所工作
- 其他规避行为

> **连续两次后，是否续签无固定期限合同的选择权仅在于劳动者**，用人单位不得拒绝。

### 18.4 竞业限制收紧（第13-15条）

- 劳动者**未接触**商业秘密的，竞业限制**不生效**
- 范围/地域/期限超出合理比例的，**超出部分无效**
- 在职期间竞业限制有效，不以支付补偿为生效要件
- 违反竞业限制须**返还补偿并支付违约金**（双重责任）

### 18.5 社保约定无效（第19条）

- 约定或承诺无需缴纳社保的，**一律无效**
- 未依法缴社保，劳动者可解除+主张经济补偿
- 用人单位补缴后，可向劳动者追索已支付的社保补偿

### 18.6 特殊待遇服务期（第12条）

用人单位提供落户、住房、奖励等特殊待遇并约定服务期的，劳动者提前离职须**赔偿实际损失**（非违约金），法院综合考量实际损失、过错程度、已履行年限。

## 附录一：员工违纪通知的送达程序

依据《劳动部办公厅关于通过新闻媒介通知职工回单位并对逾期不归者按自动离职或旷工处理问题的复函》，送达顺序如下：

```
第一  直接送达 → 书面交本人；本人不在交同住成年亲属签收
第二  邮寄送达 → 直接送达困难的，用挂号信/EMS，以回执日期为送达日
第三  公告送达 → 仅在前两种方式均无法送达时使用，公告30日视为送达
```

**重点**：
- 能用直接/邮寄送达却直接用公告的，**公告无效**
- 可将送达约定写入劳动合同（如"本合同列明地址为有效送达地址"）
- 微信/钉钉等电子方式可作为辅助送达手段，但仍建议以EMS为主要证据

## 附录二：独生子女奖励（企业职工）

- **山西省规定**：持有《独生子女父母光荣证》的职工，退休时由所在单位发放一次性奖励（具体标准按山西省人口和计划生育条例执行）
- 独生子女费发放**与是否提供劳动无关**，即使职工不在岗，保留劳动关系的单位仍应承担发放义务
- 独生子女一次性养老补助：山东省标准为设区的市上年度职工年平均工资×30%（山西省标准按省条例执行）
