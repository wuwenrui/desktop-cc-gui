## 功能说明

把用户给定的主题/素材，做成一套可在浏览器直接放映的 HTML 演示文稿（PPT）。

## 产出要求

- 单个 **自包含 HTML 文件**：所有 CSS 内联在 `<style>`，不引用任何外部 JS/CSS/字体/图片 CDN（离线可放映）。
- 每页一屏（`100vh`），上下/左右键或滚动翻页；含清晰的标题层级、要点列表、必要的图示用纯 CSS/SVG 表达。
- 16:9 版式，深浅主题任选其一并保持一致；正文中文字号 ≥ 20px，标题层级分明。
- 首页含标题 + 副标题/汇报人占位；末页含小结/联系方式占位。
- 文件名用主题的简短英文或拼音 + `.html`（如 `due-diligence-deck.html`）。

## 执行步骤

1. 若主题不明确，先用一句话和用户确认 PPT 主题与目标受众。
2. 规划大纲（封面 → 背景 → 核心内容分页 → 小结），逐页填充内容。
3. 生成单个 HTML 文件，**保存到当前工作区根目录**（便于在 lawhub「制作 PPT」菜单下列出、预览、发布协作批注）。
4. 完成后告知用户文件名，并提示可在 lawhub 菜单中预览或发布。

## 约束

- 不要拆成多个文件；不要依赖构建工具；不要引用外网资源。
- 内容基于用户提供的素材，缺失处用清晰的占位符标注，不要编造关键事实。
