## 功能说明

本 Skill 用于处理**催款类律师函草拟**事务。以委托人（债权人）聘请律师的身份，根据委托人与欠款方签订的合同及履行情况，按照既定格式批量出具正式催款律师函。

**适用场景**：广告发布费、货款、服务费、租金等各类合同项下逾期款项催收。

## 一、核心工作流

### Step 0：启动——明确发函需求

进入工作区路径，明确以下信息：
- 本次发函涉及的项目数量（几个欠款方）
- 每个欠款方对应的合同/订单数量
- 委托人提供的欠款说明（客户要求/需求文件）
- 参考发函格式（如有模板律师函）

### Step 1：阅读基础资料——提取关键合同信息

**必须从各合同/订单中提取以下结构化信息**：

| 信息项 | 说明 | 重要性 |
|--------|------|--------|
| 合同名称 | 完整名称，如《太原地铁2号线媒体广告发布合同》 | ★★★ |
| 合同主体 | 甲方/乙方的全称（精确到一字不差） | ★★★ |
| 签约日期/合同编号 | 如有 | ★★☆ |
| 合同标的/品牌项目 | 如"海信+苏宁max""邮储银行" | ★★★ |
| 合同总金额 | 大小写均需提取 | ★★★ |
| 广告发布/履行期间 | 起止日期 | ★★★ |
| 付款方式/付款节点 | 一次性付款/分期付款 | ★★★ |
| 已付金额 | 实际已支付金额 | ★★★ |
| 欠付金额 | 尚欠金额（=应付-已付） | ★★★ |
| 违约金条款 | 条款编号+具体比例/计算方式 | ★★★ |
| 合同争议解决条款 | 管辖法院/仲裁机构 | ★☆☆ |
| 合同份数 | 几份合同分别列出 | ★★★ |

**每份合同必须逐一核实上述信息，不得遗漏。**

### Step 2：核对履约证据

草拟律师函前，须与委托人提供的以下履约证据交叉核对：
- 广告上刊确认单/上刊照片（证明委托人已履行发布义务）
- 对账单/结算单（证明双方对欠款金额的确认）
- 付款凭证（证明已付金额和未付金额）
- 催款记录（如有，证明委托人已进行催收）

**如证据缺失，须在律师函中提示"基于委托人单方陈述"加以限定。**

### Step 3：法律分析

基于提取的合同信息，进行以下法律分析：

**3.1 合同效力分析**
- 合同是否依法成立并生效（双方签字盖章+内容不违反强制性规定）
- 委托人是否已全面履行自身义务

**3.2 违约责任分析**
- 欠款方是否已构成违约（付款期限是否已届满）
- 违约金条款的具体内容（条款编号+违约金比例+计算方式）
- 违约金上限问题：《民法典合同编通则司法解释》第65条——约定的违约金超过造成损失的30%的，人民法院可以予以适当减少。实务中，多数法院以LPR的1-4倍为参考，按日1‰的违约金标准（年化36.5%）明显偏高，在诉讼中可能被调减。律师函中可按合同约定主张，但需提示委托人该标准在诉讼中可能被调减的风险。

**3.3 欠款金额的精确计算**
- 每份合同分别列明：合同金额→已付金额→尚欠金额
- 多份合同的，分别列明后加总累计欠付金额

### Step 4：按格式草拟律师函

**标准格式结构**（严格按以下顺序，不得自行调整结构）：

```
律师函（标题居中，加粗大字）

【导语】{{律所}}（以下简称"本所"）接受{{公司}}（以下简称"委托人"）的委托，就向贵公司催收逾期广告发布费及违约金相关事宜，向贵公司致函如下：

一、委托人向本所陈述如下事实：
【逐一陈述各合同签订情况、履行情况、欠款情况】

【多份合同分别以"第X份合同"列明，最后汇总欠款总额】

二、根据委托人陈述的前述事实，本所律师认为：
【法律分析：合同效力+履行情况+违约责任】

三、基于前述事实和理由，委托人授权本所律师，郑重向贵公司函告如下：
1.贵公司在收到本函后5个工作日内向委托人一次性支付逾期广告发布费共计人民币【总金额】；
2.若贵公司未在上述期间内付清全部款项，委托人将采取必要之法律手段追究贵公司的法律责任。届时，贵公司除需全额支付欠款外，还需依法承担违约金及委托人为实现债权支出的各项诉讼维权成本。

特此函告。

{{律所}}
                        律师：
                                二〇二六年X月X日
```

### Step 5：风险防控——律师函的自我保护

**5.1 事实陈述的限定**
- 第一部分"委托人向本所陈述如下事实"中，必须使用"委托人陈述""根据委托人提供的信息"等限定性表述
- 本所不对欠款方未提供/未披露的信息负责
- 如存在证据不完全的情况，在律师函中明确说明"基于委托人单方陈述"

**5.2 法律分析的限定**
- 分析基于现行有效法律法规
- 如法律发生变化，意见可能需要调整

**5.3 应告知委托人的风险提示**
- 律师函并非诉讼，不具有强制执行力
- 如欠款方在5日期限内仍不支付，需要通过诉讼/仲裁途径解决
- 合同按日1‰的违约金标准（年化36.5%）在诉讼中可能被法院调减
- 注意诉讼时效（3年，自知道或应当知道权利受损之日起计算）

### Step 6：交付律师函（输出）

**输出要求**：
- 文件格式：.docx（Word文档）
- 文件命名：致【公司简称】律师函【日期】.docx
- 保存路径：同一工作区内
- 输出前再次核对欠款金额、公司名称、日期等关键信息

## 二、不同合同类型对应的文本处理要点

### 2.1 广告发布合同（太原地铁2号线媒体广告发布合同模板）
- 标准违约金条款：第九条第2款"每逾期一日，按应付未付广告费的1‰承担违约责任"
- 合同份数较多时，每份作为单独段落列明，最后汇总

### 2.2 户外画面类广告发布合同（苏宁类模板）
- 合同+订单模式：主合同为框架合同，通过媒体投放订单/补充协议确认具体发布安排
- 违约金条款可能不同，需从合同中提取违约金条款的具体内容和条款编号
- 如合同中无具体违约金比例，改用一般性逾期违约责任表述

### 2.3 其他服务/买卖合同
- 根据具体合同的违约金条款调整第二部分引用内容
- 如无明确违约金约定，引用《民法典》第577条（违约责任）、第584条（损失赔偿额）

## 三、法律依据速查

| 场景 | 法律依据 |
|------|----------|
| 合同效力 | 《民法典》第502条：依法成立的合同，自成立时生效 |
| 全面履行 | 《民法典》第509条：当事人应当按照约定全面履行自己的义务 |
| 付款义务 | 《民法典》第626条：买受人应当按照约定的数额和支付方式支付价款 |
| 违约责任 | 《民法典》第577条：当事人不履行合同义务或者履行合同义务不符合约定的，应当承担继续履行、采取补救措施或者赔偿损失等违约责任 |
| 违约金 | 《民法典》第585条：约定的违约金过分高于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以适当减少 |
| 违约金调减标准 | 《民法典合同编通则司法解释》第65条：当事人主张约定的违约金过高请求予以适当减少的，人民法院应当以实际损失为基础，兼顾合同的履行情况、当事人的过错程度以及预期利益等综合因素，根据公平原则和诚实信用原则予以衡量，并作出裁决。约定的违约金超过造成损失的百分之三十的，人民法院一般可以认定为过分高于造成的损失 |
| 诉讼时效 | 《民法典》第188条：向人民法院请求保护民事权利的诉讼时效期间为三年 |
| 维权费用承担 | 《民法典》第584条：损失赔偿额应当相当于因违约所造成的损失，包括合同履行后可以获得的利益 |

## 四、批量发函场景下的工作流

### 4.1 同时发函多个欠款方
当一次性向多个欠款方发函时：

1. **统一阅读基础资料**：一次性阅读所有项目资料，提取结构化信息
2. **逐一草拟**：每个欠款方分别草拟，遵守同一格式标准
3. **统一输出**：所有律师函保存至同一目录
4. **交叉复核**：批量发函时容易出现信息错配（如A公司的欠款金额写入B公司的函件），必须交叉复核

### 4.2 同一欠款方有多份合同的律师函
- 第一部分：说明签订了X份合同
- 中间段落：分"第X份合同"逐一列明发布时间、金额、欠款金额
- 汇总段：累加所有欠款金额

## 六、注意事项

1. **合同相对性原则**：律师函只向合同相对方发送。即使欠款方的股东、关联公司等可能实际受益，也不能在律师函中直接向他们主张权利。
2. **欠款金额精确性**：金额是最敏感的信息，必须反复核对。建议制表核对：合同金额 - 已付金额 = 尚欠金额。
3. **公司名称准确性**：欠款方全称必须在律师函中一字不差地写对，建议对照营业执照/合同签署页核实。
4. **违约金过高的处理**：合同按日1‰的违约金标准年化36.5%，远超目前司法实践中普遍支持的利率标准。律师函中按合同约定主张，但需口头提示委托人如进入诉讼，法院可能调减。
5. **证据保留**：律师函发出后，应保留快递底单（EMS或顺丰）、签收回单、律师函副本，以便后续诉讼作为催收证据。
6. **诉讼时效**：发函催收同时构成诉讼时效中断。建议在律师函发出后关注欠款方的回应，如5-7日内无回复，及时启动下一轮催收程序。
7. **日期格式**：必须使用中文大写数字，如"二〇二六年五月二十六日"（〇不是"零"也不是"0"）。
8. **语言风格**：正式、专业、克制，避免情绪化用语。律师函是法律文书，不是吵架信。
